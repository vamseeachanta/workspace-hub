## Submitting Feedback

### Claude Code Feedback

Use the `/feedback` command in Claude Code to submit feature requests directly to Anthropic.

### Workspace-Hub Improvements

1. Create a markdown file in this directory
2. Use naming convention: `<feature-name>-feature-request.md` or `<topic>-feedback.md`
3. Include:
   - Problem description
   - Proposed solution
   - Use cases
   - Alternatives considered

### Template

```markdown
# Feature Request: [Title]

## Problem
[Describe the current limitation or issue]

## Proposed Solution
[Describe your proposed improvement]

## Use Cases
- [Use case 1]
- [Use case 2]

## Alternatives Considered
- [Alternative 1 and why it doesn't work]
```
