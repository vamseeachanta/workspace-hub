## Quick Start for AI Agents

All AI agents (Claude, OpenAI, Factory.ai) must:

1. **READ** [AI_AGENT_GUIDELINES.md](AI_AGENT_GUIDELINES.md) first
2. **FOLLOW** the interactive engagement pattern
3. **ASK** clarifying questions before implementing
