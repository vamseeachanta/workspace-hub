## Conversion Process

### For Each Agent:

1. Read agent file
2. Extract frontmatter (name, description, capabilities, hooks, tools)
3. Convert to SKILL_TEMPLATE_v2.md format
4. Add new sections: Quick Start, When to Use, Configuration, Examples
5. Create skill directory and SKILL.md
6. Update documentation index

### Parallel Execution Strategy

- Use 30 parallel agents (32 threads - 2)
- Batch by category for efficiency
- Each agent handles 2-3 conversions
