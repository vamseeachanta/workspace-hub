Objective

Review the comprehensive-learning skill and define specific, measurable outcomes that will automatically roll into the /today skill report.

Context

The comprehensive-learning skill helps agents synthesize knowledge, but its outputs are currently siloed. By integrating its key findings into the /today report, we create a unified daily summary of both work completed (WRK items) and new insights gained.

Plan

  1. Review the current structure and outputs of the comprehensive-learning skill.
  2. Define a standard JSON or Markdown format for "Learning Outcomes" (e.g., key concepts grasped, unresolved questions, cross-domain links).
  3. Update the /today script (scripts/reports/generate-today.sh or similar) to parse these learning outcomes and append them to the daily summary.

Acceptance Criteria

  • [ ] comprehensive-learning skill updated to output a standardized summary artifact.
  • [ ] /today command successfully reads the artifact and includes a "Learning Outcomes" section in its output.
