arch(comp-learning): per-workstation analysis with ace-linux-1 final compilation

Problem

The current comprehensive-learning skill has a hard single-machine guard that limits the full pipeline to ace-linux-1 only. With logs/orchestrator/ now established as the unified per-machine log directory (WRK-683), every workstation should run its own Phase 1 analysis against its local logs, commit derived state, and let ace-linux-1 do the final cross-machine compilation.

Design

Each workstation runs its own comprehensive-learning analysis:
