# /performance-report - Marketing Performance Report

Generate a marketing performance report with key metrics, trend analysis, insights, and optimization recommendations.

## Usage

```
/performance-report [scope] [time period]
```

## Supported Report Types

- **Campaign-specific** analysis
- **Channel performance** (email, social, paid, SEO)
- **Content performance** tracking
- **Cross-channel** marketing summary
- **Custom** user-defined scope

## Workflow

### 1. Data Collection

Gather metrics from connected analytics tools. If not available, request user-provided data via spreadsheets, CSV files, or dashboard screenshots.

### 2. Report Structure

1. **Executive Summary** (3-5 sentences)
2. **Key Metrics Dashboard** with status indicators (on track / at risk / off track)
3. **Trend Analysis** with period-over-period comparisons
4. **Performance Wins** -- what worked and why
5. **Improvement Areas** -- what underperformed and hypotheses
6. **Data-Driven Insights** -- correlations and patterns
7. **Recommendations** prioritized by impact vs. effort
8. **Budget vs. Actual** (if applicable)

### 3. Stakeholder Formatting

Adapt output for:
- Executive-level summaries suitable for leadership
- Detailed analyst-level breakdowns with appendices

## Follow-Up Options

- Create slide deck from report
- Draft stakeholder communications
- Deep-dive into specific metrics
- Build reusable reporting template
