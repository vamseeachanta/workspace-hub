# /seo-audit - SEO Health Assessment

Perform a comprehensive SEO audit examining keyword opportunities, on-page optimization, content gaps, technical infrastructure, and competitive positioning.

## Usage

```
/seo-audit <URL or domain> [audit type]
```

## Audit Types

- **Full site audit** -- complete SEO review
- **Keyword research** -- identify ranking opportunities
- **Content gap analysis** -- find missing topics
- **Technical SEO check** -- crawlability and performance issues
- **Competitor SEO comparison** -- benchmarking analysis

## Evaluation Areas

- Title tags and meta descriptions
- Heading structure (H1-H6)
- Keyword placement and density
- Internal linking architecture
- Image optimization and alt text
- Page speed and Core Web Vitals
- Mobile responsiveness
- Schema markup
- Crawlability and indexation
- Backlink profile

## Output

1. **Executive Summary**: Overall SEO health score and key findings
2. **Keyword Opportunities**: Ranked by search volume and difficulty
3. **On-Page Issues**: Prioritized by severity (high/medium/low)
4. **Content Gap Recommendations**: Topics with effort estimates
5. **Technical Checklist**: Issues with remediation steps
6. **Competitor Comparison**: Head-to-head benchmarking
7. **Action Plan**: Quick wins (this week) and strategic initiatives (this quarter)

## Follow-Up Options

- Draft content briefs for identified gaps
- Optimize meta elements for priority pages
- Build content calendar based on keyword opportunities
- Analyze additional competitor domains
