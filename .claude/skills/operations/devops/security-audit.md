# Security Audit Skill

Invoke: `/security-audit` or load agent `@.claude/agent-library/devops/security-audit.md`

## Quick Commands
- `/security-audit scan` - Scan for vulnerabilities
- `/security-audit secrets` - Check secrets management
- `/security-audit compliance <standard>` - Check compliance

## Handoff to Agent
```
Task: [vulnerability scan | secrets audit | compliance check]
Scope: [codebase | infrastructure | dependencies]
Standard: [OWASP | SOC2 | HIPAA | PCI-DSS]
```
