# Linux (`hardware-assess.sh`) (+1)

## Linux (`hardware-assess.sh`)


| Flag | Description |
|------|-------------|
| `-o`, `--output FILE` | Override output file path |
| `-p`, `--pretty` | Pretty-print JSON (uses jq or python3) |
| `-q`, `--quiet` | Suppress log messages |
| `-h`, `--help` | Show usage |

| Flag | Description |
|------|-------------|
