# Observability Skill

Invoke: `/observability` or load agent `@.claude/agent-library/devops/observability.md`

## Quick Commands
- `/observability logs <service>` - Setup logging
- `/observability metrics <app>` - Add metrics
- `/observability trace` - Setup distributed tracing

## Handoff to Agent
```
Task: [logging | metrics | tracing | alerting]
Stack: [OpenTelemetry | Prometheus | Datadog | ELK]
Scale: [dev | staging | production]
```
