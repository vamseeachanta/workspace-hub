# Installation

## Installation


```bash
# Using pip
pip install numpy scipy sympy matplotlib

# Using UV (faster)
uv pip install numpy scipy sympy matplotlib

# Specific versions
pip install numpy==1.26.0 scipy==1.11.0 sympy==1.12
```
