# Metrics

## Metrics


| Metric | Typical Value |
|--------|---------------|
| Indexing speed | ~1000 documents/minute |
| Search latency | <50ms for 100K chunks |
| Storage overhead | ~10-20% over raw text |
| FTS5 index size | ~30% of text size |
| Memory usage | ~100MB per 50K documents |
