# Performance Tips

## Performance Tips


1. **Use Boost module** for datasets >5k points
2. **Disable animations** for real-time updates
3. **Limit tooltip calculations**
4. **Use turboThreshold** wisely
5. **Optimize data format** (avoid objects when arrays suffice)
