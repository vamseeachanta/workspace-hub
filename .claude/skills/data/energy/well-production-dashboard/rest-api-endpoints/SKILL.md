# REST API Endpoints

## REST API Endpoints


| Endpoint | Method | Description |
|----------|--------|-------------|
| `/api/health` | GET | Health check |
| `/api/wells` | GET | List all wells |
| `/api/wells/<well_id>` | GET | Get well data |
| `/api/dashboard/data` | GET | Full dashboard data |
