# Best Practices

## Best Practices


1. **Use set -e** - Exit on errors
2. **Source libraries** - Don't duplicate code
3. **Use functions** - Modular, testable code
4. **Handle signals** - Cleanup on exit
5. **Validate inputs** - Check before executing
