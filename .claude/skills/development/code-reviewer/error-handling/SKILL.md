# Error Handling

## Error Handling


| Situation | Action |
|-----------|--------|
| Too large PR | Request split into smaller PRs |
| Missing context | Ask author for explanation |
| Unclear requirements | Defer to requirements review |
| Disagreement | Escalate with evidence |
