# Error Handling

## Error Handling


| Situation | Action |
|-----------|--------|
| Unclear requirements | Return to brainstorming skill |
| Task too complex | Split into smaller tasks |
| Dependency discovered | Add prerequisite task |
| Plan exceeds scope | Document out-of-scope items |
