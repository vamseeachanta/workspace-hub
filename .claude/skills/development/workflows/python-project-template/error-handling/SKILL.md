# Error Handling

## Error Handling


### Directory Exists


```
Error: Directory 'my-project' already exists

Options:
1. Use different name
2. Use --force to overwrite
3. Use --update to add missing files
```
### UV Not Installed


```
Error: UV package manager not found

Install UV:
curl -LsSf https://astral.sh/uv/install.sh | sh
```
