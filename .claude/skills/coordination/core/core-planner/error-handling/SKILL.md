# Error Handling

## Error Handling


| Issue | Recovery |
|-------|----------|
| Blocked task | Identify alternative path |
| Agent unavailable | Reassign or wait |
| Dependency failure | Re-plan affected tasks |
| Timeline slip | Adjust estimates, re-prioritize |
