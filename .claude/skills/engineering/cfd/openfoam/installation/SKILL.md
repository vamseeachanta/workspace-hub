# Installation

## Installation


OpenFOAM v2312 is installed at `/usr/lib/openfoam/openfoam2312/` via ESI apt repo.

```bash
# Source environment before any OpenFOAM command
source /usr/lib/openfoam/openfoam2312/etc/bashrc
```
