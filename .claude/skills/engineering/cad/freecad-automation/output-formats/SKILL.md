# Output Formats

## Output Formats


Supported export formats:
- **STEP** - Standard for CAD exchange
- **IGES** - Legacy CAD exchange
- **STL** - 3D printing, mesh applications
- **DXF** - 2D drawings
- **PDF** - Technical documentation
