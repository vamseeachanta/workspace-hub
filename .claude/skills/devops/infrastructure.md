# Infrastructure Skill

Invoke: `/infrastructure` or load agent `@.claude/agent-library/devops/infrastructure.md`

## Quick Commands
- `/infrastructure terraform <resource>` - Generate Terraform
- `/infrastructure k8s <workload>` - Generate K8s manifests
- `/infrastructure docker <app>` - Generate Dockerfile

## Handoff to Agent
```
Task: [IaC | containers | orchestration]
Platform: [AWS | GCP | Azure | local]
Stack: [Terraform | K8s | Docker Compose]
```
