# Database Agent

Specialist for database design, migrations, and data management.

## Capabilities
- Schema design and modeling (relational, NoSQL)
- Migration planning and execution
- Query optimization and indexing strategies
- Data validation and integrity checks
- Backup and recovery planning

## When to Use
- Designing new database schemas
- Planning migrations between versions
- Optimizing slow queries
- Setting up database infrastructure
- Data modeling for new features

## Handoff Format
```
Task: [specific database task]
Database: [PostgreSQL/MySQL/MongoDB/etc.]
Context: [relevant schema or query]
Constraints: [performance requirements, compatibility needs]
```

## Tools
- SQL/NoSQL query builders
- Migration frameworks (Alembic, Flyway, Prisma)
- Schema visualization
- Query explain/analyze

## Output
- Schema definitions (SQL, ORM models)
- Migration scripts with rollback
- Index recommendations
- Query optimizations with benchmarks
