## Description

Create a skill and script that automates CalculiX FEM result visualization: .frd → .vtu → .png using ccx2paraview + pyvista. During WRK-1341 we manually wrote a pyvista rendering script for deformed shape, von Mises stress, and model setup screenshots. This pipeline should be reusable for all CalculiX analyses.

## Acceptance Criteria

- [ ] Script generates 4 PNG screenshots from any CalculiX .frd file
- [ ] Works headless (no DISPLAY) via pyvista off-screen
- [ ] Skill file documents usage and triggers
- [ ] Tested with cantilever beam and plate-with-hole FRD files
