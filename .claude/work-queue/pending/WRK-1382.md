## Mission

Create the definitive naval architecture expert skill — an AI agent that represents the best naval architect who is simultaneously an engineering and legal consultant. This is the capstone of the naval architecture knowledge extraction pipeline. The agent embodies the combined expertise of ~30 textbooks, 110 ship plans, and regulatory standards.

## What

### Engineering Consultant Persona
1. **Design advisor**: hull form selection, powering estimates, stability assessment
2. **Calculation engine**: routes to digitalmodel functions for hydrostatics, resistance, propulsion, seakeeping, stability
3. **Source-traced answers**: every recommendation cites textbook + page + equation number
4. **Trade-off analysis**: articulates engineering trade-offs (e.g., stability vs speed, draft vs displacement)
5. **Historical reference**: 110 ship plans as precedent library — "vessels of similar type used X"

### Legal Consultant Persona
1. **Regulatory compliance**: automated IMO/SOLAS/ABS/DNV checks against vessel parameters
2. **Classification guidance**: which class rules apply, what scantlings are required
3. **Liability awareness**: flags when a design parameter is near regulatory limits
4. **Documentation**: generates survey-ready calculation reports with regulatory references
5. **Jurisdiction mapping**: knows which flag state rules apply to which vessel types

### Knowledge Foundation
- 7,127+ classified records (constants, equations, procedures, definitions, requirements)
- 33 EN400 verified worked examples + 100+ deep-extracted examples
- 3,724 regulatory requirements from SOLAS, ABS, DNV, MCA
- 1,826 engineering constants with units and sources
- 815 equations with derivation context
- Ship-specific hydrostatic tables (DDG-51, FFG-7, etc.)
- GZ curves and stability data

## Acceptance Criteria

1. Skill file created with dual persona (engineering + legal)
2. Topic routing covers all naval arch disciplines
3. Agent can perform calculations via digitalmodel and cite sources
4. Agent can check regulatory compliance and flag concerns
5. Agent can reference historical ship data as design precedent
6. All responses include source citations (textbook, page, standard clause)
7. Legal deny-list compliance verified (no client references)
8. Cross-review by Codex agent passes
