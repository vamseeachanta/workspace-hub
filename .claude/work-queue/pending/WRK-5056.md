# feat(digitalmodel): integrate theoretical naval architecture into ecosystem models

## Mission
Review the promoted naval architecture content from WRK-5054 and integrate the theoretical/analytical content into digitalmodel's engineering modules: resistance prediction, stability analysis, and hull form parameterization.

## Why
WRK-5054 extracted 1826 constants, 815 equations, 3724 requirements, and 248 definitions from 34 textbooks and hydrostatics references. The theoretical plans (Attwood 1899, Comstock 1942, PNA series, USNA EN400) contain analytical methods that belong in digitalmodel's computation modules, not just as promoted raw text.

## Key Textbooks with Theoretical Content
- Theoretical Naval Architecture (Attwood 1899) — classical theory
- Introduction to Naval Architecture (Comstock 1942) — foundational
- PNA Second Revision Vol 1-3 — comprehensive theory (stability, resistance, motions)
- USNA EN400 Principles of Ship Performance — modern academic treatment
- Marine Hydrodynamics (Newman 2018) — advanced hydrodynamics theory
- Offshore Hydromechanics (2001) — hydrostatics/dynamics reference

## Approach
1. Review promoted constants.py — curate into typed, validated constants module
2. Implement key equations as Python functions with proper docstrings:
   - ITTC 1957 friction line
   - Holtrop-Mennen resistance estimation (if extractable)
   - IMO intact stability criteria checks
   - Hydrostatic calculations (KB, BM, GM from hull geometry)
3. Wire promoted worked examples into pytest validation
4. Integrate with existing hydrodynamics module in digitalmodel

## Acceptance Criteria
- [ ] Naval architecture constants module with validated, typed values
- [ ] At least 5 analytical functions implemented and tested
- [ ] IMO stability criteria checker function
- [ ] Integration tests against known textbook examples
