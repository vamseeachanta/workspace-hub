## Mission

Prepare a self-conducted cost segregation study for 15645 Westpark Drive (Family Dollar #30150) using published retail benchmark percentages. Produce a defensible component breakdown with documented methodology, suitable for Form 4562 filing.

## What

- Research published cost segregation benchmarks for single-tenant NNN retail properties
- Apply benchmark percentages to the $1,089,534 building basis
- Classify components into 5-year, 7-year, 15-year, and 39-year MACRS asset classes
- Compute year 1 depreciation with 40% bonus depreciation (2025 TCJA phase-down)
- Document methodology with IRS references for audit defensibility
- Produce final component schedule for Form 4562

## Key Inputs

- Building basis: $1,089,534.35 (from tax-preparation-worksheet.yaml)
- Placed in service: September 2025
- Property type: Single-tenant commercial retail (Family Dollar), modified NNN lease
- Lot: 1.5628 acres (68,076 sqft)
- HCAD account: 111-299-000-0073
- Bonus depreciation rate: 40% (2025)

## Methodology Approach

1. Collect published benchmark data from:
   - AICPA cost segregation audit technique guide
   - IRS Cost Segregation Audit Techniques Guide (2022)
   - Published studies on retail/dollar store properties
2. Use conservative end of benchmark ranges (lower reclassification %)
3. Document each component category with IRS asset class reference (Rev. Proc. 87-56)
4. Cross-reference against HCAD improvement/land split for reasonableness

## Expected Output

- `sabithaandkrishnaestates/docs/tax/2025-cost-segregation-study.yaml` — component schedule
- `sabithaandkrishnaestates/docs/tax/2025-cost-segregation-methodology.md` — methodology narrative with references

## Acceptance Criteria

1. Component breakdown totals equal building basis ($1,089,534.35)
2. Reclassification percentage is conservative (≤30% of building to shorter lives)
3. Each component category cites IRS asset class and Rev. Proc. 87-56 reference
4. Methodology document references published benchmarks with source citations
5. Final depreciation schedule ready for Form 4562 line items
