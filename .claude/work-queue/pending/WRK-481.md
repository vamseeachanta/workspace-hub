# feat(digitalmodel): integrate GEBCO_2025 Bathymetry for subsea routing

## Objective
Integrate the GEBCO_2025 15-arc-second bathymetry grid into subsea pipeline routing and installation planning.

## Context
GEBCO_2025 is the latest global bathymetric dataset. Programmatic access via OPeNDAP enables water depth lookups for any offshore location, directly supporting routing and site assessment tasks.

## Plan
1. Implement OPeNDAP client for GEBCO_2025 data extraction.
2. Integrate depth lookups into subsea pipeline routing modules.
3. Provide water depth annotation for asset coordinates.
4. Verify accuracy against existing local bathymetry data if available.

## Acceptance Criteria
- Programmatic water depth retrieval functional.
- Routing modules correctly utilize GEBCO bathymetry for water depth constraints.
