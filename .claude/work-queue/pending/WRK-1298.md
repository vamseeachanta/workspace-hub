## Mission

OCR 39K scanned PDFs on ace-linux-2 using GPU-accelerated tools (NVIDIA T400 4GB).
This is the ace-linux-2 shard of Phase 2b from WRK-1245.

## Context

- ace-linux-2 has NVIDIA T400 (4GB) — can run doctr or tesseract with GPU
- 39,127 ocr-needed PDFs are local on /mnt/dde/
- ace-linux-1 has 52,915 ocr-needed but GTX 750 Ti (2GB) — less capable for OCR

## Scope

- Benchmark 3 OCR tools on 50-doc sample: tesseract, doctr (GPU), azure-doc-intelligence
- Select best tool based on accuracy/speed/cost tradeoff
- Run OCR on 39K scanned PDFs, output text layers
- Re-run readability classification on OCR'd docs

## Acceptance Criteria

1. [ ] OCR tool benchmarked on 50-doc sample with accuracy metrics
2. [ ] ≥35K scanned PDFs OCR'd with text output
3. [ ] Re-classified as machine-readable after OCR
4. [ ] Quality spot-check on 20 random docs
