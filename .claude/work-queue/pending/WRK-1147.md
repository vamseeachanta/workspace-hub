## Mission

Research, assess, and implement a calculation framework for propeller-rudder hydrodynamic interaction across the full engine operating envelope — from engine-off through RPM-driven drive-off — suitable for ship maneuvering analysis.

## What / Why

When a ship manoeuvres, the rudder operates in the hydrodynamic field produced by the propeller. The propeller's state — locked, free-wheeling, or rotating at varying RPM — fundamentally changes the velocity field at the rudder plane, altering rudder forces and effectiveness. Capturing this interaction requires:

- Solid grounding in the available literature (semi-empirical, momentum-based, vortex, and RANS-derived methods)
- A structured method comparison to select the best approach for a fast-running Python model
- An implementation parameterised by RPM, delivered power, and ship speed — covering the full drive-off transient (n = 0 → rated)

This underpins future maneuvering performance modules in `digitalmodel`.

## Acceptance Criteria

- [ ] Python module implementing selected method(s) with RPM sweep (engine-off → rated RPM)
- [ ] Worked example with representative ship parameters; results as plots/tables
