## Mission

Run deterministic deep extraction on 125K machine-readable PDFs local to ace-linux-2
(/mnt/dde/ NTFS disk). This is the ace-linux-2 shard of Phase 2a from WRK-1245.

## Context — WRK-1277 learnings

- dde_project files are at `/mnt/dde/` on ace-linux-2 (NTFS mounted)
- Path remap: `/mnt/remote/ace-linux-2/dde/` → `/mnt/dde/`
- Must use pdftotext (poppler), not pdfplumber — NTFS I/O can cause D-state hangs
- 8-worker multiprocessing with subprocess timeout is the proven pattern (WRK-1277)

## Scope

- Run deep extraction on 125K machine-readable PDFs on ace-linux-2 local NTFS
- Use pdftotext for text, subprocess timeout for safety
- Extract: tables, charts metadata, constants, equations
- Copy results back to ace-linux-1 for integration
- Estimated runtime: ~30h at 1.2 docs/sec

## Acceptance Criteria

1. [ ] Deep extraction run on ≥120K ace-linux-2 machine PDFs
2. [ ] Table extraction yield ≥65% on dde_project segment
3. [ ] Results copied back to ace-linux-1
4. [ ] Extraction errors categorized and logged
