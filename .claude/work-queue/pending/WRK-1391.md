## Mission

Update the `freecad-automation` skill with patterns learned during WRK-1360 implementation.

## What

1. Add bend arc pattern (`_make_bent_tube`) to the FreeCAD skill's capabilities — 3-point arc sweep for hollow tubes with configurable CLR
2. Document coordinate system convention (origin at shared junction, vehicle-centric axes)
3. Add `is_bend` / `bend_clr` member metadata pattern to skill's geometry templates
4. Document mandrel bend radius standards (2D/3D/4D) for 4130 chromoly tubing
5. Add STEP/IGES export with bend geometry to skill examples

## Why

The FreeCAD skill currently has no bend arc support. WRK-1360 developed this capability ad-hoc. Capturing it in the skill makes it reusable for future structural frame projects.

## Acceptance Criteria

- [ ] `freecad-automation` skill references bend arc sweep pattern
- [ ] Coordinate convention documented in skill
- [ ] Bend radius table (2D/3D/4D) included
- [ ] Example shows hollow tube with arc sweep
