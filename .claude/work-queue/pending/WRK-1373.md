## Mission

Build frequency-domain 6-DOF motion prediction using extracted knowledge from Newman Marine Hydrodynamics, SNAME Vol III, and Bertram Practical Ship Hydrodynamics. Currently zero seakeeping capability exists in digitalmodel.

## What

1. RAO-based heave/pitch/roll response in regular waves
2. Spectral integration for irregular sea response (JONSWAP, PM spectra already in hydrodynamics/)
3. Operability index calculation against sea state scatter diagrams
4. Motion sickness incidence (MSI) at specified locations
5. Natural period estimation (heave, pitch, roll)

## Reference Sources

- Newman Marine Hydrodynamics (2018) — Ch 6-7: added mass, damping, wave forces
- SNAME PNA Vol III — Ch 8: motions in waves
- Bertram Practical Ship Hydrodynamics — Ch 4: seakeeping
- EN400 Ch 8 — 1 worked example (basic encounter frequency)

## Acceptance Criteria

1. Heave, pitch, roll RAO computation from vessel data
2. Spectral response calculation
3. Natural period estimation functions
4. TDD tests from textbook worked examples
5. Integration with existing hydrodynamics/ wave spectra
