# feat(worldenergydata): create Offshore Geohazard Feed module using USGS API

## Objective
Implement a geohazard monitoring module using the USGS Earthquake Hazards FDSN API.

## Context
Seismic risk is critical for offshore infrastructure. The USGS API provides free, real-time GeoJSON feeds that can be queried by geographic bounding box.

## Plan
1. Create `worldenergydata/safety_analysis/geohazard` module.
2. Implement USGS FDSN API wrapper.
3. Add functionality to query seismic events around specific offshore assets.
4. Integrate with geospatial mapping if available.

## Acceptance Criteria
- Real-time seismic event retrieval functional.
- Radius-based queries around asset coordinates return correct historical and recent events.
