## Mission

Create a naval architecture expert skill and agent configuration that combines:
- 7,127+ extracted knowledge records (constants, equations, procedures, definitions, requirements)
- Curated TDD fixtures from 30 textbooks
- digitalmodel calculation modules
- Ship-specific data tables
- Regulatory requirements database

The agent should be able to answer naval architecture questions, perform calculations, cite sources, and generate code with engineering justification.

## What

1. **Expert skill** at `.claude/skills/digitalmodel/naval-architecture/SKILL.md`
   - Topic routing: hydrostatics, stability, resistance, propulsion, seakeeping, structures
   - Knowledge index pointers: which JSONL/YAML to query per topic
   - Calculation function registry: what digitalmodel can compute
   - Source citation protocol: always cite textbook + page

2. **Agent configuration** in AGENTS.md
   - naval-architecture-expert subagent type
   - Entry reads: knowledge indexes, fixture files, module API
   - Tools: digitalmodel functions, doc-intelligence query

3. **Knowledge query skill**
   - Search constants, equations, procedures by keyword
   - Return source-traced results with page numbers
   - Cross-reference multiple textbooks on same topic

4. **Calculation agent**
   - Route calculation requests to appropriate digitalmodel function
   - Validate inputs against known ranges
   - Report results with uncertainty and source basis

## Acceptance Criteria

1. Skill file created with topic routing
2. Agent can answer "what is the metacentric height formula?" with source citation
3. Agent can compute GZ curve for a given vessel using digitalmodel
4. Agent can check IMO stability criteria
5. Knowledge query returns results from multiple textbooks
6. All responses include textbook/standard citations
