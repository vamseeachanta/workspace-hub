## Mission

Implement hull form geometry and parametric design using extracted knowledge from PNA Vol I, Tupper, and Rawson-Tupper. Enable hull definition from principal particulars.

## What

1. Form coefficients: block (Cb), prismatic (Cp), waterplane (Cw), midship (Cm)
2. Sectional area curve generation from coefficients
3. Series 60 standard series interpolation
4. Lackenby transformation for hull variants
5. Displacement, LCB, LCF from offset tables

## Reference Sources

- SNAME PNA Vol I — hull geometry
- Tupper Introduction to Naval Architecture — Ch 2-3
- EN400 Ch 2 — 6 worked examples (hull form)

## Acceptance Criteria

1. Form coefficient calculations validated against EN400 examples
2. Sectional area curve from offsets
3. Series 60 parent hull lookup
4. TDD tests from textbook data
