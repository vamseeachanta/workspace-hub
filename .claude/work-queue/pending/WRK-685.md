# feat(skills): update /today to highlight availability of unused preview AI models

## Objective
Update the `/today` work item/report to actively check for and display the availability of new or "preview" AI models that the agent harness can access but are not currently being utilized in regular sessions.

## Context
As providers (Anthropic, OpenAI, Google) release new preview models, we want the system to proactively alert us so we can test them against the orchestrator gates. Integrating this check into the `/today` report ensures it gets daily visibility.

## Plan
1. Identify the config file or API endpoint where the agent harness registers available models (e.g., `.claude/config/ai-tools/agent-quota-latest.json` or `.codex/settings.json`).
2. Write a script to compare the "available" models against the "actively configured" models in the `AGENTS.md` and related config files.
3. If a preview model is available but unused, append a "New AI Capabilities Available" section to the `/today` output.

## Acceptance Criteria
- [ ] `/today` report includes a section identifying any configured but currently unused preview models.
- [ ] The check runs quickly without adding significant latency to the report generation.
