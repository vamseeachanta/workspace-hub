## What

Create OrcaFlex template and automation for parachute deployment dynamics — model the chute as a line object with time-dependent drag, capture snap loads and deceleration profile.

## Why

WRK-5082 Child 6 (OrcaFlex dynamic analysis) needs a parachute deployment model. OrcaFlex's cable dynamics engine is ideal for capturing transient snap loads during deployment. Execution on acma-ansys05 (OrcaFlex license).

## Scope

1. Define parachute as OrcaFlex line type (Cd, projected area, mass)
2. Model deployment as velocity-dependent drag with time ramp
3. Set up implicit time-domain solver (capture snap transients)
4. Extract: time-domain forces at frame attachment, deceleration profile, peak snap load
5. Compare peak dynamic load with static drag force (dynamic amplification factor)

## Acceptance Criteria

- [ ] OrcaFlex model runs for 200 MPH and 250 MPH deployment cases
- [ ] Time-domain force history at attachment point extracted
- [ ] Peak snap load quantified vs static drag (DAF reported)
- [ ] Deceleration profile (g vs time) plotted
- [ ] Results exported to YAML for integration with WRK-5082 pipeline
