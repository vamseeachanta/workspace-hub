## What

Two related issues from WRK-1181 download script execution:

1. **Metocean DEST path**: download-literature.sh uses `$(git rev-parse --show-toplevel)/../worldenergydata/...` instead of absolute `/mnt/ace/worldenergydata/...`. Files downloaded to wrong location; had to be manually copied.

2. **set -e incompatibility**: All 10 download scripts had `set -euo pipefail` which aborts on first download failure. Scripts were patched to `set -uo pipefail` during WRK-1181 but the template should also be updated.

## Acceptance Criteria

1. Metocean script uses absolute `/mnt/ace/worldenergydata/...` path
2. `download-template.sh` uses `set -uo pipefail` (not `set -euo`)
3. All 10 existing download scripts confirmed to use `set -uo pipefail`
4. Dry-run of metocean script writes to correct `/mnt/ace/` path
