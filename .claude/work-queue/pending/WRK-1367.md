## Mission

Perform member capacity and connection design checks — unity ratios for all members, bolts, welds, and pins under governing load cases.

## What

Using forces from the governing analysis model (per WRK-1366 recommendation):

### Member Checks
1. **Axial capacity**: tension yield, compression buckling (Euler/Johnson) for each tube member
2. **Bending capacity**: plastic moment vs applied moment
3. **Combined**: interaction equation (axial + bending) per AISC 360 H1.1 or equivalent
4. **Unity ratios**: demand/capacity for each member, each load case

### Connection Checks
1. **C1 bolt+pin**: bolt shear, bearing on plate, pin bending/shear
2. **B1 bolted (6x)**: bolt group shear, prying action, bearing
3. **Coupler pin**: double-shear capacity, pin bending between strut eyes
4. **Welds (C0, C2, C3)**: fillet weld capacity per load direction (longitudinal/transverse shear)

### Material
- 4130 chromoly: Fy=460 MPa, Fu=560 MPa (normalized condition)
- Weld: assume E70xx filler (matching or overmatching)

## Why

Unity ratios are the deliverable that answers the client's question: is the frame adequate for 200/250 MPH deployment? Any ratio >1.0 identifies members or connections that need reinforcement.

## Acceptance Criteria

- [ ] Unity ratios computed for all members under all load cases
- [ ] Unity ratios computed for all connections (C1, B1, coupler, C0, C2, C3)
- [ ] All ratios < 1.0 for assumed dimensions (or deficiencies flagged)
- [ ] Governing member and governing connection identified
- [ ] Checks use forces from both 2D and 3D analyses (extended with 3D FEM per decomposition)
- [ ] Results tabulated in YAML for report integration (WRK-1368)
