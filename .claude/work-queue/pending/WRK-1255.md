## Mission

Batch deep extraction on ~1,500 machine-readable ace_standards PDFs >1MB.
These are full engineering standards (API RPs, DNV RPs, ABS guides) — the richest
documents per-doc with avg 36 tables and 22 figures each.

## Why

WRK-1246 yield assessment: ace_standards >1MB have 53% table rate, 52% figure rate,
mean 36.3 tables/doc. Extrapolated ~69K tables from just 1,899 docs. These are the
highest-value standards for methodology extraction (WRK-1245 Phase 3).

Of the 1,899 docs, ~77% are machine-readable (extract directly), ~16% need OCR (skip
for now), ~7% mixed. Focus on the ~1,500 machine-readable ones.

## Plan

### Step 1: Filter machine-readable standards

Use the readability enrichment (WRK-1246 background job) or re-classify the 1,899
docs inline. Select only `readability=machine` docs.

### Step 2: Build batch extraction script

Create `scripts/data/doc-intelligence/batch-extract-standards.py`:
- Filter: source=ace_standards, ext=pdf, size_mb>=1.0, readability=machine
- Run `extract_document()` → manifest → `deep_extract_manifest()` per file
- 60s timeout per file (these are large), skip >50MB
- Write: tables → CSV, charts → metadata YAML, extraction report
- Append to manifest-index.jsonl
- Progress log every 10 files (each takes 5-15s)

### Step 3: Run batch extraction

```bash
nohup uv run --no-project python scripts/data/doc-intelligence/batch-extract-standards.py \
    > logs/enrichment/batch-extract-standards-$(date +%Y%m%d).log 2>&1 &
```

Estimated: ~1,500 docs × 10s avg = ~4-5 hours.

### Step 4: Promote high-value tables

Identify and promote tables containing:
- S-N curves, fatigue design factors
- Material properties, allowable stresses
- Environmental load coefficients
- Corrosion protection design tables

## Acceptance Criteria

1. [ ] Batch script with timeout + resume support
2. [ ] ~1,500 machine-readable standards processed
3. [ ] Tables/charts exported to `data/doc-intelligence/batch/standards/`
4. [ ] Manifest index updated
5. [ ] Summary: total tables, figures, top-10 richest documents identified
6. [ ] ≥20 high-value tables promoted to `data/standards/promoted/`
