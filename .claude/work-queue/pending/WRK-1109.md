## Mission

Install `lharries/whatsapp-mcp` on ace-linux-1, connect via QR code, and verify Claude Desktop can send/receive WhatsApp messages.

## What

Set up the most popular WhatsApp MCP server (5,400+ stars) on ace-linux-1 for personal use with Claude Desktop.

## Why

WRK-1108 research confirmed WhatsApp MCP integration is available now. This WRK implements it.

## Acceptance Criteria

- [ ] Go and uv installed on ace-linux-1 (verify prerequisites)
- [ ] `lharries/whatsapp-mcp` cloned and built
- [ ] Claude Desktop config updated with MCP server entry
- [ ] QR-code auth completed — phone linked
- [ ] Smoke test: Claude can list chats and send a test message
- [ ] Document setup steps in `docs/integrations/whatsapp-mcp.md`

## Notes

- Repo: https://github.com/lharries/whatsapp-mcp
- Uses unofficial whatsmeow Go library — do not use for high-volume automated messaging (ToS risk)
- Prompt injection risk: be cautious exposing received message content to Claude in untrusted contexts
- If unstable, fall back to `delltrak/wamcp` (Docker-based, 63 tools, also supports official Meta Cloud API)
