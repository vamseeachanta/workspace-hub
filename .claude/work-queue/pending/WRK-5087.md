## Mission

Build an autoresearch-style overnight loop that proposes, evaluates, and keeps/reverts SKILL.md improvements.

## What

Create `scripts/cron/skill-autoresearch-nightly.sh` following the karpathy/autoresearch pattern:
1. Git branch isolation (`autoresearch/skills-YYYY-MM-DD`)
2. Pick skills with low eval scores or high failure rates (from skill-executions.jsonl)
3. Claude proposes SKILL.md amendment
4. Run skill-eval — keep if score improves, revert if not
5. Log result to `results.tsv` (append-only)
6. Human reviews branch diff next morning

## Why

From WRK-5085 research: the autoresearch pattern (git as state machine, fixed-budget eval, keep/revert) maps directly to skill improvement. Running overnight compounds agent effectiveness without consuming interactive session time.

## Acceptance Criteria

- [ ] Cron script runs unattended on branch (never touches main)
- [ ] Each improvement attempt has fixed time budget (3 min max)
- [ ] Results logged to append-only TSV
- [ ] Morning review: `git diff main..autoresearch/skills-<date>`
- [ ] Integrates with existing skill-eval infrastructure
