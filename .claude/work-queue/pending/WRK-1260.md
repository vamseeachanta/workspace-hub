## Mission

Mechanical batch cleanup of skill ecosystem issues identified by WRK-1244 eval.

## What

1. Remove 36 leftover README.md files from skill directories (v2 anti-pattern)
2. Whitelist common HTML tags (`<details>`, `<summary>`, `<br>`, `<sub>`, `<sup>`)
   in `audit-skill-violations.sh` — reduces 127 violations to ~10
3. Add `scripts_exempt: true` frontmatter field for guidance-only skills (AI prompting,
   documentation tools, etc.) and update `skill-coverage-audit.sh` to skip them —
   reduces 339 coverage gaps to ~50
4. Fix 10 duplicate skill names, 38 short descriptions, 10 TODO markers

## Why

- 163 violations and 339 coverage gaps are mostly false positives
- Noisy audit output causes alert fatigue; weekly eval becomes ignored
- All fixes are mechanical / batch-scriptable

## Acceptance Criteria

- [ ] 0 README.md files in skill dirs
- [ ] HTML tag violations < 15 (real violations only)
- [ ] Coverage gaps < 60 (real gaps only)
- [ ] 0 duplicate skill names
- [ ] 0 TODO/FIXME markers in skills
