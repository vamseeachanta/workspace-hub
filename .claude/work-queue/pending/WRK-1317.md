# WRK-1317: Generalize Stage-Transition Engine to All Workflows

## Problem Statement

WRK-1316 hardens stage transitions for the work-queue lifecycle (hooks, checklists, HTML refresh, per-stage tool activation). That same discipline — declarative stages, mandatory checklists, transition hooks, HTML visibility — should apply to ALL workspace-hub workflows, not just the WRK lifecycle.

Currently, other workflows (calculation, Excel extraction, research, cross-review) are defined as prose-only skills (L0) without:
- Declarative stage definitions
- Transition hooks or checklists
- HTML progress visibility
- Per-stage tool/skill activation

## Scope

### Phase 1: Audit existing workflows
Inventory all workflows across workspace-hub skills and scripts. For each, document:
- Current stages (implicit or explicit)
- Enforcement level (L0–L3)
- Pain points and violations observed
- Candidate stages for hooks/checklists

### Phase 2: Extract reusable workflow engine from WRK-1316
- Abstract the stage-transition config YAML format
- Make hooks, checklists, HTML generation workflow-agnostic
- Create `scripts/workflow/engine.py` (or similar) that any workflow can use

### Phase 3: Create child WRKs per workflow
Each existing workflow gets its own WRK to adopt the engine:
- Calculation workflow (engineering calcs pipeline)
- Excel calculation workflow (dark intelligence extraction → validated calc)
- Research-to-WRK workflow
- Cross-review workflow (standalone, per WRK-1010 merge recommendation)
- Code porting / legal scan workflow
- Session start/end workflow
- Others discovered during Phase 1 audit

## Acceptance Criteria

- [ ] AC1: Workflow audit YAML produced listing all workflows + enforcement levels
- [ ] AC2: Reusable workflow engine extracted from WRK-1316 deliverables
- [ ] AC3: Child WRKs created for each workflow adoption
- [ ] AC4: At least one non-WRK-lifecycle workflow fully migrated to engine (proof of generality)

## Dependencies

- **WRK-1316** must complete first (provides the concrete implementation to generalize from)
