## Mission

Unlock `digitalmodel/naval_architecture/curves_of_form.py` (currently NotImplementedError) by extracting and encoding real ship hydrostatic data tables from EN400 appendices and SNAME publications.

## What

1. Extract EN400 Ship Data Section: DDG-51, FFG-7, CVN-65, AOE-6, LCS-1, YP (6 vessels)
2. Parse curves of form: displacement, KM, TPI, MT1" vs draft
3. Parse cross curves of intact stability: KN vs displacement at multiple heel angles
4. Encode as YAML data files in `digitalmodel/data/ships/`
5. Implement interpolation in `curves_of_form.py` using extracted tables
6. TDD: verify against known EN400 homework answers

## Acceptance Criteria

1. ≥ 4 vessels with complete hydrostatic tables
2. `curves_of_form.py` returns valid interpolated values
3. Tests pass against EN400 worked examples (chapters 3-4)
4. Data files version-controlled with source traceability
