## Mission
Ingest individual rig names, contractors, and detailed metadata from state railroad commissions (Texas RRC, NDIC, COGCC, etc.) and industry sources to complement Baker Hughes aggregate rig counts.

## Context
Baker Hughes free data provides aggregate counts with basin/county/trajectory metadata but no individual rig names or contractor info. State commissions publish well permits and drilling reports that often include rig contractor and rig name fields.

## Potential Sources
1. **Texas RRC** — drilling permits (Form W-1) include rig contractor
2. **North Dakota Industrial Commission (NDIC)** — daily activity reports with rig names
3. **Colorado COGCC** — well permits and scout tickets
4. **IHS / Enverus** — commercial (premium license required)
5. **Rig data aggregators** — RigData, DrillingInfo (now Enverus)

## Acceptance Criteria
1. Survey available free public sources for rig-level metadata
2. Implement loaders for at least 1 state commission (Texas RRC preferred)
3. Schema maps rig name/contractor to Baker Hughes basin/county/date
4. TDD tests passing with fixture data
5. Ingestion report YAML with source coverage assessment
