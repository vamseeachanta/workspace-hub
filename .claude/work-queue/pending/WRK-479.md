# feat(worldenergydata): wire SODIR FactPages OData API

## Objective
Expand the `worldenergydata/sodir` module to ingest the full scope of SODIR/NPD FactPages data using the OData API.

## Context
The existing SODIR module only ingests a subset of available data. The OData API provides access to 30,000+ datasets, including well data, production, seismic, and relinquished-area datasets.

## Plan
1. Audit all available SODIR FactPages/FactMaps OData endpoints.
2. Extend the module schema to support new dataset types.
3. Implement an ingestion pipeline for bulk and incremental updates.
4. Add verification against the FactPages web portal.

## Acceptance Criteria
- Module can ingest all core FactPages datasets via OData.
- Relinquished-area and seismic metadata included in the ingestion scope.
