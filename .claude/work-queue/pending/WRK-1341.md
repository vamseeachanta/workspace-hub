## What

Extend CalculiX INP writer to support beam elements (B31, B32) and frame analysis. Currently only 3D solid elements (C3D4, C3D8, etc.) are supported.

## Why

WRK-5082 Child 5 (3D FEM analysis) needs beam/frame elements in CalculiX to model the parachute frame as a beam FEM and compare with 2D direct stiffness results.

## Scope

1. Add B31, B32, T3D2 to ELEMENT_TYPE_MAP in inp_writer.py
2. Implement `add_beam_section()` with I_yy, I_zz, A, J, cross-section type
3. Write `_write_beam_sections()` for `*BEAM SECTION` cards
4. Add moment/rotation loads (DOFs 4, 5, 6)
5. Add distributed loads (`*DLOAD`)
6. Add 1D line meshing in gmsh mesh_generator.py
7. Validation: cantilever beam deflection (textbook), frame with known reactions

## Acceptance Criteria

- [ ] B31 beam element in INP file accepted by ccx v2.21
- [ ] Cantilever beam tip deflection within 1% of PL^3/3EI
- [ ] Frame analysis reactions match 2D direct stiffness solver (within 2%)
- [ ] All existing 30 CalculiX tests still pass (regression)
