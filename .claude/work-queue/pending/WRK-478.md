# feat(subsea): create Cathodic Protection module (DNV-RP-B401)

## Objective
Implement a Cathodic Protection (CP) design and verification module based on DNV-RP-B401.

## Context
WRK-383 flagged the absence of a CP module as a significant gap. Every subsea structure requires CP design verification.

## Plan
1. Create `digitalmodel/subsea/cp_analysis` module.
2. Implement CP current demand calculations per DNV-RP-B401.
3. Implement anode design logic (bracelet, flush, stand-off).
4. Add verification tests for various structure types (pipelines, manifolds).
5. (Optional) Explore FEniCSx integration for complex geometry CP modelling.

## Acceptance Criteria
- CP current demand and anode life calculations match DNV-RP-B401 benchmarks.
- Module supports standard anode types and structure materials.
