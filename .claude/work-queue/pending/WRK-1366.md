## Mission

Quantify the simplification error between 2D direct stiffness, 3D CalculiX FEM, and OrcaFlex frame analysis results.

## What

1. **Collect results** from WRK-1363 (2D), WRK-1364 (CalculiX), and WRK-1365 (OrcaFlex)
2. **Compare per member**: axial force, shear, moment — tabulate % difference
3. **Compare reactions** at each BC: magnitude and direction differences
4. **Compare deflections** at chute attachment point and critical locations
5. **Identify** which members show the largest discrepancy and explain why (out-of-plane effects, connection modeling differences, etc.)
6. **Recommend** which model governs for design checks

## Why

The 2D model is faster to iterate but makes simplifying assumptions (planar, rigid connections). Quantifying the error against 3D FEM and OrcaFlex determines whether the 2D model is adequate for design or if 3D results must govern.

## Acceptance Criteria

- [ ] Comparison table: member forces (axial, shear, moment) across all three tools
- [ ] Comparison table: reaction forces at each BC across all three tools
- [ ] Deflection comparison at chute attachment point
- [ ] % difference reported per member per quantity
- [ ] Members with >10% discrepancy flagged with root cause explanation
- [ ] Recommendation on governing model for design documented
