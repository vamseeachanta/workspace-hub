## Mission

Implement the selected propeller-rudder interaction method as a Python module parameterised by shaft RPM, delivered power, and ship speed — covering the full drive-off envelope from engine-off to rated RPM.

## What

Build a calculation module with the following structure:

```
digitalmodel/src/digitalmodel/hydrodynamics/
  propeller_rudder/
    __init__.py
    propeller_state.py      # slipstream velocity as f(n, V, D, P)
    rudder_forces.py        # lift/drag/torque as f(slipstream, rudder geometry)
    interaction.py          # combined propeller-rudder interaction
    rpm_sweep.py            # sweep n = 0 → rated, return results dataframe
```

**Operating cases handled:**
- Engine-off, locked propeller (n = 0, Jlocked)
- Engine-off, free-wheeling propeller (n = n_fw from windmilling equilibrium)
- Engine startup: low RPM, high J, lightly loaded
- Drive-off: n ramps from idle → rated; J decreases; thrust/torque increase
- Full ahead: rated RPM, design J

**Key computed quantities at each RPM step:**
- Advance ratio J = V / (n · D)
- Thrust coefficient KT, torque coefficient KQ (from open-water diagram or regression)
- Axial slipstream velocity Va at rudder plane
- Tangential slipstream velocity Vt at rudder plane
- Rudder effective inflow angle, lift force FL, drag force FD, side force Fy

## Why

This is the deliverable that makes the research actionable — a reusable, tested Python module integrated into `digitalmodel` that future maneuvering studies can call directly.

## Acceptance Criteria

- [ ] Module structured as above; importable from `digitalmodel.hydrodynamics.propeller_rudder`
- [ ] `rpm_sweep(n_rated, V, D, P_rated, rudder_params, steps=20)` returns a DataFrame with all key quantities at each RPM step
- [ ] Engine-off sub-cases (locked + free-wheeling) handled as n=0 boundary condition
- [ ] Open-water diagram input: accepts KT/KQ polynomial coefficients (Wageningen B-series or user-supplied)
- [ ] ≥5 unit tests: n=0 locked, n=0 free-wheeling, J→0 (bollard pull), J=design point, full RPM sweep shape check
- [ ] Worked example script: `examples/propeller_rudder_driveoff.py` with plots of FL, Va, J vs n/n_rated
