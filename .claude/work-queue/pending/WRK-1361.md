## Mission

Derive dimensioned 2D engineering drawings from the 3D FreeCAD model — plan, elevation, side views with all member lengths, angles, and connection details.

## What

Generate production-quality 2D views from WRK-1360's 3D CAD model:

1. Plan view (top-down) — overall frame footprint, mount point spacing
2. Rear elevation — trunk frame: horizontal bar, V-struts, coupler pin location
3. Side elevation — full load path from under-hood to rear bracket
4. Connection details — enlarged views of C1 (bolt+pin), B1 (6x bolt pattern), coupler pin
5. Dimension all members, angles, and offsets per hand sketch values

## Why

Dimensioned drawings are the reference for analysis model validation (do the FEM node coordinates match the drawing?) and for the final engineering report deliverable.

## Acceptance Criteria

- [ ] 2D drawings generated from FreeCAD TechDraw or equivalent
- [ ] All member lengths, angles, and offsets dimensioned
- [ ] Connection details shown at enlarged scale
- [ ] Drawings exportable to PDF (printable at A3/A2)
- [ ] Dimensions cross-checked against hand sketch measurements
