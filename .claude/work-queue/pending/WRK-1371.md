## Mission

The table quality audit shows 84% of 3,683 promoted CSVs are junk (watermarks, OCR artifacts). Create a curation script that flags usable tables and archives junk.

## What

1. Write `scripts/data/doc-intelligence/curate-promoted-tables.py`
2. Apply table_quality.py filters: watermark detection, content threshold (≥3 rows, ≥2 numeric cols)
3. Partition into `promoted/naval-architecture/usable/` and `promoted/naval-architecture/archived/`
4. Generate curation report with counts per source document
5. TDD: test the curation logic with sample CSVs

## Acceptance Criteria

1. Usable tables identified and separated (expected ~500 from 3,683)
2. Curation report written
3. Archived tables moved to prevent confusion
4. Tests pass
