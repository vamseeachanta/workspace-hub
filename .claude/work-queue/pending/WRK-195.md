# EIA US Production Data Module — Non-GoM Onshore + Alaska

## Why

BSEE covers the GoM OCS at well/lease level — but the US produces ~13 million
bopd total (2025) and GoM is only ~1.8 million bopd (~14%). The other 86%:
Permian Basin (~6.6M bopd, ~50% of US total), Bakken, Eagle Ford, DJ Basin,
Alaska (~0.4M bopd, growing) — is covered exclusively by EIA, not BSEE.

EIA provides the best-documented open API in the global O&G data landscape:
free REST API, consistent series IDs, state and basin-level monthly data,
and an international country-level database that covers 200+ countries. Adding
EIA completes the US picture and provides the global aggregate layer for
cross-basin analysis (GoM vs Permian vs Alaska vs international).

## Data Source

**API:** `https://api.eia.gov/v2/` (free API key, no cost)
**API documentation:** `https://www.eia.gov/opendata/`
**Bulk download:** `https://www.eia.gov/dnav/pet/pet_crd_crpdn_adc_mbbl_m.htm`

Key series:
- **EIA-914**: Monthly mandatory survey, covers top 375 producers (85%+ of US output);
  state-level and basin-level resolution; lag ~4-6 weeks
- **Crude oil production by state** (series format: `PET.MCRFPUS2.M`): all 50 states monthly
- **Alaska production**: state series + Prudhoe Bay / Kuparuk / Point Thomson breakdown
- **International**: `https://www.eia.gov/international/data/world` — country-level monthly,
  160+ countries, from EIA's own surveys + IEA inputs + national stats offices
- **Drilling productivity report**: region-level (Permian, Bakken, Eagle Ford, Anadarko,
  Appalachian, Haynesville, Niobrara) with rig count + new well productivity

## Key Data Series (EIA API v2)

| Dataset | Series concept | Use |
|---------|---------------|-----|
| US total crude production | `PET.MCRFPUS2.M` | National aggregate |
| Production by state | `PET.MCRFPXX2.M` (XX = state code) | State-level DCA |
| Alaska production | `PET.MCRFPAK2.M` | Arctic benchmark |
| GoM federal offshore | `PET.MCRFPUS2GOM.M` | Crosscheck vs BSEE |
| Permian DPR | Drilling Productivity Report | Permian rig + IP data |
| International production | `INTL.57-1-*.A` (annual) / EIA ILD (monthly) | Global context |

## Notes

- EIA API v2 (released 2023) replaces deprecated v1; use v2 exclusively
- Free API key: register at `https://www.eia.gov/opendata/register.php`
- Rate limit: 500 requests/hour per key — implement caching to stay within limit
- Shale DCA uses hyperbolic (b > 1 early, b → 0.5 terminal) not exponential —
  important distinction vs conventional fields in NCS/UKCS/Brazil modules
