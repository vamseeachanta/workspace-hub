## Mission

Search for, download, and index electrical engineering reference documents into the shared document-intelligence standards library at `/mnt/ace/docs/_standards/Electrical-Engineering/`.

## What

### Seed URLs (starting points)

1. **MIT OpenCourseWare — Circuits and Electronics** — https://ocw.mit.edu/courses/6-002-circuits-and-electronics-spring-2007/
2. **MIT OpenCourseWare — Electrical Engineering and Computer Science** — https://ocw.mit.edu/course-lists/electrical-engineering-and-computer-science/
3. **All About Circuits — Textbooks** — https://www.allaboutcircuits.com/textbook/
4. **IEEE Xplore Open Access** — https://ieeexplore.ieee.org/browse/books/title
5. **NFPA 70 (NEC) free access resources** — search for freely available NEC study guides and summaries
6. **Electronics Tutorials** — https://www.electronics-tutorials.ws/

### Execution steps

1. Fetch seed URLs — extract all linked PDFs and further resource links
2. Search online for additional freely-available electrical engineering PDFs/textbooks (IEEE, IEC, NEC, power systems, circuit theory, etc.)
3. Download all available PDFs to `/mnt/ace/docs/_standards/Electrical-Engineering/`
4. Catalogue each download: title, author, year, source URL, local path
5. Add catalogue entries to `knowledge/seeds/electrical-engineering-resources.yaml`
6. Regenerate document-index so files are searchable

## Why

Build a local offline library of electrical engineering references to support future module development (power systems, instrumentation, control systems) in the workspace-hub ecosystem.

## Acceptance Criteria

- [ ] All seed URLs fetched and linked PDFs extracted
- [ ] Web search performed for additional freely-available resources
- [ ] All downloadable PDFs saved to `/mnt/ace/docs/_standards/Electrical-Engineering/`
- [ ] Catalogue YAML written at `knowledge/seeds/electrical-engineering-resources.yaml`
- [ ] Document index regenerated (or update queued)
- [ ] No paywalled or DRM-protected content downloaded
