## Mission

Take deep-extracted worked examples (from WRK-1369) and curate them into per-topic TDD fixture files for digitalmodel. Verify numerical values against source PDFs where confidence is low.

## What

1. Filter examples with `use_as_test: true` from extraction output
2. Group by topic: hydrostatics, stability, resistance, propulsion, seakeeping, submarines
3. Create fixture YAML files in `digitalmodel/tests/fixtures/naval-architecture/`
4. Verify at least 20 examples by spot-checking against source PDFs
5. Promote to parameterized pytest files via `promote-to-code.py`

## Acceptance Criteria

1. ≥ 80 curated fixture entries across ≥ 5 topics
2. Fixture YAML files created per topic
3. Promoted test files generated and runnable (xfail where implementation pending)
4. Spot-check report documenting verification of 20+ examples
