## Mission

Evaluate Zellij (Rust-based terminal multiplexer with WebAssembly plugin system)
for adoption across the workspace-hub workstation ecosystem.

## What

- Install and test Zellij on ace-linux-1 (primary dev machine)
- Assess key features: layout persistence, floating/stacked panes, session management
- Compare with current tmux/screen usage (if any)
- Evaluate cross-platform support: Linux (ace-linux-1/2), macOS (MacBook Air), Windows (aceengineer-va, acma-ansys05)
- Test integration with Claude Code terminal workflow
- Document install steps per workstation in workstations skill

## Why

Zellij offers session persistence, collaborative multiplayer, and a WebAssembly
plugin system that could improve multi-agent and multi-session workflows across
the workstation fleet.
