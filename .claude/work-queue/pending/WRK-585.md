# Add IRENA Offshore Wind Capacity Data to Energy Mix Module

## What

Integrate IRENASTAT offshore wind capacity and generation data into the
worldenergydata energy mix module.

## Why

IRENA provides authoritative global renewable energy statistics. Offshore
wind is a growing segment adjacent to the O&G modules. Adding this data
enables energy transition analysis alongside existing hydrocarbon modules.

## Deliverables

- IRENASTAT data access method documented (API or download)
- Data integrated into energy mix module
- Test coverage for data ingestion

## Acceptance Criteria

- [ ] IRENASTAT data source identified and access method documented
- [ ] Offshore wind capacity data extracted
- [ ] Integrated into worldenergydata module
- [ ] Tests covering data format validation
