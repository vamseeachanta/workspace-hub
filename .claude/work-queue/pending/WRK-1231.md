## Mission
Create a CompanyLoader class for BSEE company data. Binary files exist (`companydetails/mv_companies_active.bin`, `mv_companies_all.bin`) and download URL is registered, but no loader provides programmatic DataFrame access.

## Context
Gap identified during WRK-1189 resource intelligence. Every other data category (well, platform, pipeline, rig fleet) has a loader except company/operator hierarchy.

## Acceptance Criteria
1. `CompanyLoader` class in `bsee/data/loaders/company/` following PlatformLoader pattern
2. Methods: `load_active()`, `load_all()`, `get_by_operator_id()`, `get_by_company_name()`
3. Returns standardized DataFrames
4. TDD tests with fixture data (15+ tests)
5. Ingestion report YAML with record counts
