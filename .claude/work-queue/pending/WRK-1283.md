## Mission

Review rudder type classification videos, catalogue the different rudder types
(balanced, semi-balanced, spade, flap, Schilling, Becker, etc.), and assess which
types can be modelled within the digitalmodel hydrodynamics module.

## What

- Watch and extract rudder type taxonomy from the three reference videos
- Map each type to hydrodynamic characteristics (lift slope, stall angle, drag)
- Identify which types the current Söding + AD model (WRK-1149) can handle
- Document gaps requiring additional models (e.g. flap rudders, high-lift devices)
- Output: docs/domains/hydrodynamics/rudder-types-survey.md

## Why

The WRK-1149 method selection assumed a conventional single-piece rudder. Different
rudder types have different lift/drag characteristics that affect the interaction
model accuracy and applicability.
