## Mission
Research and identify a Linux-compatible speech-to-text tool similar to Whisper-flow that enables voice input for terminal/AI agent sessions.

## What
Find a Linux equivalent to Whisper-flow — a real-time or near-real-time speech-to-text tool that integrates with the desktop workflow on dev-primary.

## Why
Voice input can accelerate AI agent interactions, reducing typing friction for long prompts and commands.

## Acceptance Criteria
- [ ] Identify 2-3 candidate tools (e.g., whisper-live, whisper-jax, nerd-dictation, asr-hotkey)
- [ ] Compare: latency, accuracy, GPU/CPU requirements, Linux GNOME integration
- [ ] Confirm compatibility with dev-primary hardware
- [ ] Document recommended tool and installation path
- [ ] Capture findings as a WRK knowledge artifact
