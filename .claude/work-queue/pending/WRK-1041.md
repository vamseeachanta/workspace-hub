# Lifecycle HTML — auto-refresh + on-demand /work html command

## Mission
**Add auto-refresh to generated lifecycle HTML and a /work html WRK-NNN on-demand regeneration command so the browser view stays current without manual reloads.**

## What

1. Auto-refresh: embed <meta http-equiv="refresh" content="30"> in generated HTML.
2. On-demand: /work html WRK-NNN regenerates lifecycle HTML; --open flag triggers xdg-open.

## Why

Stage-start/end scripts will regenerate HTML at each boundary. Without auto-refresh,
the user must manually reload — causing stale views and context-switching errors.

## Acceptance Criteria
- [ ] generate-html-review.py --lifecycle output includes meta refresh tag (30s)
- [ ] /work html WRK-NNN command added to work-queue SKILL and implemented
- [ ] --open flag triggers xdg-open after regeneration
- [ ] Existing HTML generation tests pass
