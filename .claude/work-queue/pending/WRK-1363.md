## Mission

Extend `frame_solver.py` with the full frame geometry (rear trunk + under-hood) and solve for member forces, reactions, and deflections under parachute drag loading.

## What

The existing `frame_model.py`/`frame_solver.py` models only the **rear trunk frame** (Page 1: horizontal bar + V-struts, 6 nodes, 6 members). This child must:

1. **Add under-hood frame members** (Page 2 geometry): horizontal bar, curved member at 150°, B1 bolted connection (6 bolts), P1 pinned BC
2. **Connect both assemblies** through the vehicle chassis load path (C3 → chassis → B1)
3. **Apply drag force** from WRK-1362 at the chute attachment point (coupler pin)
4. **Solve** using direct stiffness method for all load cases:
   - Case 1: Single chute at 200 MPH
   - Case 2: Single chute at 250 MPH
   - Case 3: Dual chute at 250 MPH (if applicable per WRK-1362 findings)
5. **Extract**: member axial forces, shear, moments; reaction forces at all BCs; deflections

## Why

The current model overestimates fixity at C3 (treats trunk endpoints as fully fixed) because the under-hood frame flexibility is missing. Adding it gives realistic reaction distributions and deflections for the member/connection checks (WRK-1367).

## Acceptance Criteria

- [ ] Frame model includes both rear trunk and under-hood assemblies
- [ ] Node coordinates match WRK-1360 CAD geometry (within 0.25")
- [ ] Member forces and reactions computed for all load cases
- [ ] Equilibrium verified: sum of reactions = applied drag force (within 0.1%)
- [ ] Deflections reported at chute attachment point and all connections
- [ ] Results exported to YAML for comparison (WRK-1366) and checks (WRK-1367)
