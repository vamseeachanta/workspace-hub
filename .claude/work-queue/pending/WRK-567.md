# chore(wrk): Back-link Calc Examples to Enhancement WRK Items WRK-482..559

## What

Once WRK-560..565 are complete (calc_examples added to data-source YAMLs),
add a `test_data:` front-matter field to each relevant enhancement WRK item
(WRK-482..559) that lists the specific calc files to use as test fixtures.

This creates a direct, traceable link:
**gap WRK item → calc example files → unit test fixtures**

## Why

When an agent executes WRK-497 (implement DNV-OS-F101 pressure containment),
it should immediately know which Excel files on disk contain reference calculations
to validate against — without manually searching the index.
The `test_data:` field in the WRK item front-matter provides this without any
extra lookup.

## Steps

1. Wait for WRK-560..565 to be complete (calc_examples sections exist in YAMLs)
2. For each WRK-482..559 that has a `Standard` section in its body:
   - Match the standard ID to entries in the relevant `calc_examples` section
   - Add `test_data:` block to that WRK item's YAML front-matter, e.g.:
     ```yaml
     test_data:
     - path: /mnt/ace/docs/disciplines/.../0116-CAL-0001-01 List of Pipelines Installed.xlsx
       description: Pipeline wall thickness — reference for DNV OS F101 checks
       standard: DNV-OS-F101
     ```
3. Regenerate `INDEX.md` after all back-links added
4. Optionally: build a script `scripts/data/document-index/back-link-wrk-calcs.py` to automate this

## Scope

- WRK-482..496: cathodic-protection (15 items)
- WRK-497..538: pipeline — digitalmodel + doris (42 items)
- WRK-539..554: structural — digitalmodel + OGManufacturing (16 items)
- WRK-555..559: marine (5 items)
