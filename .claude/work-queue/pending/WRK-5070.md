## Mission

Research, assess, and implement calculation frameworks for metocean engineering — covering wave statistics, extreme value analysis, joint probability methods, wind/current modelling, and design criteria derivation — suitable for fast-running assessment and integration with the NDBC/metocean data modules.

## What / Why

Metocean analysis provides the environmental design basis for all offshore structures. Existing code spans `worldenergydata/src/worldenergydata/metocean/` (12 files — NDBC analysis, wave/metocean data) and `digitalmodel/docs/domains/metocean/`. This study will:

- **Wave statistics** — significant wave height, peak/zero-crossing period, spectral width, wave steepness
- **Extreme value analysis** — Gumbel, Weibull, GEV distributions; return period estimation; threshold selection (POT)
- **Joint probability** — Hs-Tp contours, environmental contour methods (IFORM, direct sampling), conditional distributions
- **Wind modelling** — wind profile (power law, log law), gust factors, turbulence intensity, API RP 2A wind
- **Current modelling** — depth profiles, tidal/residual decomposition, loop current (GoM)
- **Design criteria** — 100-year return values, associated values, combined loading cases
