## Mission

Systematically replace all Windows/Git Bash-incompatible patterns (`bc`, `python3`,
`nproc`) across `scripts/` and `.claude/hooks/` using canonical patterns from
`.claude/skills/_core/bash/cross-platform-compat/SKILL.md`.

## What

Audit (2026-03-11) found 199+ incompatibilities across workspace-hub scripts.
WRK-1117 fixes the learning pipeline specifically. This WRK sweeps the rest.

| Pattern | Count | Severity | Impact |
|---------|-------|----------|--------|
| `bc` / `bc -l` | 55+ | HIGH | Breaks on Git Bash |
| `python3` command | 97 | HIGH | Not in PATH on Windows |
| `nproc` | 2 | HIGH | Not available on Git Bash |
| `sed -i` no backup | 41 | MEDIUM | macOS only (deferred scope) |

### Replacement Reference

Use patterns from `.claude/skills/_core/bash/cross-platform-compat/SKILL.md`:
- `bc` → `awk "BEGIN {print ($a < $b) ? 1 : 0}"`
- `python3` → `$PYTHON` via `resolve_python()` helper
- `nproc` → `nproc 2>/dev/null || getconf _NPROCESSORS_ONLN ... || echo 1`
