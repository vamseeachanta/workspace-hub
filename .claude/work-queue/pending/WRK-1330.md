## Mission

Extract strategic value from 409 archived WRK items so they become pure audit trail — all learnings live in mission docs, roadmap, and knowledge base.

## Two Phases

### Phase 1 (Option B): One-time synthesis script
- Backfill `capture-wrk-summary.sh` for all 409 archived items (only 21 currently captured)
- Group by category (engineering, harness, data, etc.)
- Extract unaddressed `future-work.yaml` follow-ons
- Generate per-repo/category "lessons + open threads" summary
- Update `roadmap-2026-h1.md` with discovered themes

### Phase 2 (Option C): GitHub Issues as durable layer going forward
- Ensure archived WRKs with unfinished follow-ons have open GitHub Issues
- Close GitHub issues only when follow-on is truly done
- Roadmap reads from open issues, not WRK archive

## Acceptance Criteria
- [ ] All 409 archived WRKs have knowledge-base entries
- [ ] Synthesis report grouped by category exists
- [ ] Unaddressed future-work items surfaced as GitHub Issues
- [ ] Roadmap updated with themes from archive analysis
- [ ] Future archive flow automatically creates GitHub Issues for follow-ons
