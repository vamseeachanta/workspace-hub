## Mission

Research, assess, and implement calculation frameworks for asset integrity and fitness-for-service (FFS) assessment — covering metal loss, crack-like flaws, creep, dent-gouge combinations, and remaining life — per API 579-1/ASME FFS-1 and BS 7910.

## What / Why

Asset integrity assessment determines whether degraded equipment can continue to operate safely. Existing `digitalmodel/src/digitalmodel/asset_integrity/` has API 579, fracture mechanics, and RSF calculation modules. This study will:

- **Metal loss assessment** — Level 1/2/3 per API 579 Part 4/5; remaining strength factor (RSF)
- **Crack-like flaws** — Failure Assessment Diagram (FAD) per API 579 Part 9 / BS 7910; stress intensity factors
- **Fracture mechanics** — K solutions, J-integral, reference stress method, Paris law fatigue crack growth
- **Creep assessment** — API 579 Part 10; remaining life estimation, Larson-Miller parameter
- **Dent-gouge interaction** — API 579 Part 12; pipeline-specific assessment
- **Hydrogen and environmental effects** — HIC/SOHIC, SCC, corrosion fatigue

Ground existing implementations in standards and validate against published worked examples.
