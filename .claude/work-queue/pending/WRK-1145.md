# feat(charts): Anthropic-Style Research Chart Library

## Mission

Build reusable chart components (micro-skills + Python scripts) modeled on the
Anthropic "Labor Market Impacts of AI" appendix chart patterns, for use in
workspace-hub analysis reports (AI exposure, worldenergydata trends, asset analytics).

## Source Reference

Appendix to "Labor market impacts of AI" (March 2026, Anthropic)
PDF: https://cdn.sanity.io/files/4zrzovbb/website/e5f77fc0e77c0185110b5e4b909602791ae76eae.pdf
Blog: https://www.anthropic.com/research/labor-market-impacts

## Chart Types to Implement

### Chart 1: Dual-Panel Time Series + DiD Coefficient (Appendix Figures 1 & 3)

**Structure:**
- Top panel: Two-series line chart (treated group = red, control = blue), 2016–present
- Vertical dashed line = event marker (e.g. ChatGPT release, product launch)
- Bottom panel: DiD coefficient over time with translucent confidence interval shading
- Pooled post-period estimate annotation in bottom-right corner
- Shared x-axis, y-axis labeled per panel

**Use cases in workspace-hub:**
- AI exposure vs. non-exposure time trends (WRK-1128 observed-exposure tracker)
- Metric before/after a system change with statistical evidence
- Any intervention analysis (engineering reliability, financial performance)

**Script:** `scripts/charts/dual_panel_did.py`

### Chart 2: Sensitivity / Robustness Error-Bar Plot (Appendix Figure 2)

**Structure:**
- X-axis: continuous threshold or parameter (percentile cutoff, hyperparameter value)
- Y-axis: pooled effect estimate (ATT, R², correlation)
- Connected dots with vertical 95% CI bars
- Horizontal reference line = baseline threshold (dotted, labeled)
- Clean minimal grid

**Use cases:**
- Sensitivity of analysis to threshold choice (e.g. gate coverage %, exposure cutoff)
- Model performance vs. regularization parameter
- Robustness checks for any regression analysis

**Script:** `scripts/charts/sensitivity_errorbars.py`

### Chart 3: Lower-Triangular Correlation Heatmap (Appendix Figure 4)

**Structure:**
- Lower triangle only (upper triangle grey/blank)
- Color scale: deep green (ρ ≈ 1.0) → yellow (ρ ≈ 0.7) → red (ρ ≈ 0.5)
- Cell values annotated (2 decimal places)
- Row/column labels at diagonal
- Clean white background, no gridlines

**Use cases:**
- Comparing multiple measures/models (exposure metrics, financial ratios)
- Feature correlation in ML pipelines (assethold, worldenergydata)
- Cross-provider agreement matrix (Claude vs. Codex vs. Gemini scoring)

**Script:** `scripts/charts/lower_tri_heatmap.py`

## Design Principles (from Anthropic appendix)

- Color coding: red = exposed/primary series, blue = comparison/control
- Translucent shading for confidence intervals (alpha ≈ 0.2)
- Event markers: vertical dashed lines with `\n<label>` text below x-axis
- Captions below figures with figure number and data source
- Academic style: minimal chrome, generous whitespace, Seaborn or matplotlib
- Consistent font: serif for body, sans-serif for axis labels
