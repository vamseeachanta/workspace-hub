# chore(review): Troubleshoot submit-to-{claude,codex,gemini}.sh + Comprehensive Test Suite

## Why

The submit scripts are called by agents (Claude, Codex, Gemini) during cross-review
gates. A failure in any of these scripts causes a gate failure that blocks the entire
WRK lifecycle. Current issues and any new ones discovered during WRK-1002/1003/1004
runs need to be systematically identified, fixed, and guarded by a test suite so
cross-review success rates improve for all three providers.

This item is intentionally open-ended: it collects issues as they are discovered
across WRK-1002/1003/1004 runs and treats them as a consolidated fix+test batch.

**I-06 confirmed failure (real run, 2026-03-03):**
```
# Claude review failed or timed out (exit 1)
# Root cause: provider invocation did not return renderable structured output.
# Transport mode: isolated temp directory + print/json schema
# File: .claude/work-queue/assets/WRK-624/workflow-governance-review.html
```
Root cause: `workflow-governance-review.html` (30 KB) is written raw to the temp
content file. Claude reads raw `<div>`, `<style>`, `<script>` tags and fails to
produce the required JSON schema output. No HTML-to-text conversion happens anywhere
in the pipeline.

**I-06 fix:** detect `.html` / `.htm` extension before writing the content file;
strip tags to plain text first:
```bash
if [[ "$CONTENT_FILE" =~ \.(html?|htm)$ ]]; then
  # Strip HTML tags; preserve text content for review
  CONTENT="$(sed 's/<[^>]*>//g' <<< "$CONTENT" | sed '/^[[:space:]]*$/d')"
fi
```
Or use `python3 -c "import html, re, sys; ..."` via `uv run --no-project python`
for a more robust conversion. Add `--strip-html` flag to allow explicit override.

Add rows to this table as WRK-1002/1003/1004 execution surfaces new failures.

Apply fixes for every confirmed issue. For each fix:

**I-01 fix (python3 → uv):**
```bash
# Before
python3 "$BUNDLE_BUILDER" --input "$CONTENT_FILE"
python3 "$RENDERER" --provider claude --input "$raw_file"

# After
uv run --no-project python "$BUNDLE_BUILDER" --input "$CONTENT_FILE"
uv run --no-project python "$RENDERER" --provider claude --input "$raw_file"
```

**I-02 fix (setsid fallback):**
```bash
# Replace hard-exit with warn+continue
if command -v "${SETSID_CMD:-setsid}" >/dev/null 2>&1; then
  SETSID_CMD="${SETSID_CMD:-setsid}"
else
  echo "WARN: setsid not found — running without process-group isolation" >&2
  SETSID_CMD=""
fi
# In run_claude_once: ${SETSID_CMD:+"$SETSID_CMD"} timeout ...
```
