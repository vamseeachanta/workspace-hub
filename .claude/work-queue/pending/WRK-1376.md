## Mission

Implement Holtrop-Mennen (1984) statistical resistance prediction method. Currently only ITTC 1957 friction line exists. Holtrop-Mennen is the industry standard for preliminary powering.

## What

1. Total resistance decomposition: frictional + form factor + wave + appendage + correlation
2. Form factor (1+k1) from hull parameters
3. Wave resistance coefficient from Holtrop regression
4. Appendage drag for rudder, bilge keels, shaft brackets
5. Power prediction: EHP → DHP → BHP chain

## Reference Sources

- Holtrop & Mennen (1982, 1984) original papers
- SNAME PNA Vol II — Ch 5: resistance
- Bertram — Ch 3: resistance
- EN400 Ch 7 — 4 worked examples (basic scaling)

## Acceptance Criteria

1. Holtrop-Mennen resistance prediction for displacement hulls
2. Validated against published regression test cases
3. TDD from textbook worked examples
4. Integration with existing Reynolds/Froude functions
