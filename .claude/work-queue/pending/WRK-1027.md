# WRK-1027 — DNV-RP-B401 Real-World Benchmark Validation

## Mission
Add parametric pytest benchmarks that verify the B401-2021 module against three
real-world CP designs sourced from BSEE public data and internal calculation notes.

## What
The existing 114 tests in WRK-478 cover unit and integration cases with synthetic
inputs. This WRK adds end-to-end benchmark tests using real engineering inputs and
hand-verified expected outputs to confirm the module produces industry-correct results
at the system level.

## Why
Generic unit tests confirm algorithmic correctness but do not catch systematic
constants errors (wrong current density table values, wrong coating category constants)
that would still pass synthetic-input tests. Real-world benchmarks with hand-calced
ground-truth catch these class of errors.

## Sources
1. **BSEE GOM shallow-water jacket** (internet source)
   - `https://www.bsee.gov/stats-facts/offshore-information` + BOEM field data
   - 4-pile fixed jacket, ~70m water depth, Gulf of Mexico tropical
2. **FST hull coastal LNG** (document intelligence: calc-010)
   - `digitalmodel/docs/domains/cathodic_protection/examples/calc-010-dnv-b401-2021-fst-design-philosophy.md`
   - Floating Storage Terminal, brackish water (1–32 ppt seasonal), 40yr design life
3. **Deepwater subsea RBS** (document intelligence: calc-004)
   - `digitalmodel/docs/domains/cathodic_protection/examples/calc-004-dnvgl-b401-subsea-riser-base.md`
   - 4-RBS group, 1710–1900m water depth, 27yr design life, multi-anode-type

## Hand-Calculated Ground Truth
(Full working in `.claude/work-queue/assets/WRK-478/wrk-478-real-world-examples-notes.md`)

### Case 1 — GOM Jacket (25°C, ρ=0.20 Ω·m, Cat I, 25yr, Al stand-off)
- I_mean = 114.6 A; I_final = 210.1 A
- R_anode = 0.1077 Ω (Dwight, L=1.0m, r=0.05m)
- N_mass = 74; N_current = 91; recommended = 91 (current governs)

### Case 2 — FST Hull (10°C, ρ=0.5875 Ω·m, 40yr, Al flush)
- f_cm = 0.12; f_cf = 0.22 (k=0.005/yr, f_ci=0.02)
- I_mean = 322.7 A (using 200 mA/m² bare steel current density)
- Anode mass = 66,515 kg; N = 2,419 (mass governs)
- Note: ABS GN Ships current density differs from B401 Table 3-1;
  parametric test should use the B401 submerged density for the module
  and document the mapping.

### Case 3 — Deepwater RBS (0.31 Ω·m, 27yr, SO-90 stand-off)
- SO-90: R_a_init = 0.119 Ω → I_per_anode = 2.10 A
- FM-65: R_a_init = 0.273 Ω → I_per_anode = 0.916 A
- Total 4-RBS demand: 9.29 A; total mass required: 679.65 kg
