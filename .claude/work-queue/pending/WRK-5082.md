## Mission

Calculate the force in the parachute frame from the launch/deployment of a parachute on a drag car.

## What

Determine the structural loads on the parachute mounting frame during deployment, using drag car parachute deployment dynamics.

## Objective 1 — Load Distribution Through Strut Frame

Assess how the parachute drag load transfers from the chute attachment point through the strut frame members to the vehicle mounting points. This must be understood before stress checks.

Key questions to resolve:
- Load path from chute line → bracket → tubular arm → triangulated frame → body mount points
- Which members are in tension, compression, or bending at each load case
- Reaction forces at each body mount bolt location
- Whether the frame behaves as a truss (axial only) or frame (axial + bending)

## Acceptance Criteria

- [ ] **Load path mapped**: chute → bracket → arm → frame → body mounts, with force type per member
- [ ] Parachute drag force calculated for both load cases (200 MPH, 250 MPH)
- [ ] Reaction forces at each body mount point quantified
- [ ] Frame member forces resolved (axial, shear, bending) at each joint
- [ ] Stress check per von Mises yield criterion
- [ ] Stress check per ASME combined stress criteria
- [ ] Unity checks (demand/capacity) reported for each critical member and connection
- [ ] Bolt/connection adequacy verified at mounting points
