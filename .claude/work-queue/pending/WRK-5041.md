# feat(doc-intel): extract-url.py — internet document extraction pipeline

## Mission
Create scripts/data/doc-intelligence/extract-url.py that fetches URLs (web pages + online PDFs), assigns doc-ref from URL hash + title, runs same extraction pipeline as extract-document.py, adds to index.jsonl.

## Why
Engineering standards and references exist online as well as in local files. URL extraction extends the pipeline to internet sources while reusing the same manifest format.

## Acceptance Criteria
- [ ] extract-url.py accepts URL and produces extraction manifest
- [ ] Handles HTML web pages (text + table extraction)
- [ ] Handles online PDFs (download + pdfplumber parse)
- [ ] doc-ref generated from URL hash + page title
- [ ] Output manifest compatible with build-doc-intelligence.py
- [ ] Respects robots.txt and rate limiting
- [ ] Caches downloaded files to avoid re-fetching

## TDD Approach
Test URL parsing and doc-ref generation. Test HTML extraction with fixture HTML. Test PDF download handling. Test manifest compatibility with indexer.
