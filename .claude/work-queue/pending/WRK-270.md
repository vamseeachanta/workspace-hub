# Fix cathodic-protection SKILL.md — Align Examples to Real API

## What

The `cathodic-protection` SKILL.md (v1.0.0) contains Python examples referencing classes
that do not exist in the codebase: `AnodeDesign`, `PipelineCP`, `CoatingAnalysis`,
`ICCPDesign`, `CPMonitoring`, `SurveyAnalysis`, `InterferenceAnalysis`. Anyone following
these examples will get `ImportError`.

The actual API is a config-dict driven router pattern via `CathodicProtection().router(cfg)`.

## Why

A skill with broken examples actively misleads — an agent following the skill will write
code that fails immediately. This is worse than no skill.

## Plan

- **Files**: `.claude/skills/engineering/marine-offshore/cathodic-protection/SKILL.md`

  patterns matching the actual `cathodic_protection.py` API. Use the existing test in
  `tests/specialized/cathodic_protection/test_abs_cathodic_protection_calcs.py` as the
  reference for correct input structure.
