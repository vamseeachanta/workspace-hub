## Mission

Define and enforce a canonical file/directory structure for WRK item assets across all 20 lifecycle stages, derived from observed patterns in session logs.

## What

Analyze existing session logs and WRK asset directories to identify the actual file structure patterns that have emerged organically. Codify the best patterns into a canonical anatomy spec within the work-queue-workflow skill, and enforce conformance at stage entry/exit.

## Why

WRK asset directories (`.claude/work-queue/assets/WRK-NNN/`) currently have inconsistent structure — different sessions create evidence files, checkpoints, HTML, and cross-review outputs in varying locations and naming conventions. A canonical anatomy makes scripts more reliable, cross-review easier, and onboarding faster.

## Approach

1. **Audit session logs** — scan all session logs to catalog how files are saved per stage (filenames, paths, formats)
2. **Identify patterns** — extract the dominant structure, flag divergences
3. **Define canonical anatomy** — document the required directory layout and file naming per stage in work-queue-workflow SKILL.md (or a referenced spec)
4. **Add enforcement** — update `start_stage.py` / `exit_stage.py` or create a new validation script to check conformance
5. **TDD/Evals** — write tests and eval cases that verify:
   - Stage exit produces expected files in expected locations
   - Validation script catches missing/misplaced files
   - Existing compliant WRKs pass, non-compliant ones are flagged
6. **Update skills** — roll findings into work-queue, work-queue-workflow, and any other affected skills

## Acceptance Criteria

- [ ] Audit of session logs completed with pattern catalog
- [ ] Canonical file anatomy documented (per-stage required/optional files)
- [ ] Enforcement script or validation added to stage exit pipeline
- [ ] TDD tests written and passing for anatomy validation
- [ ] Eval cases covering compliant and non-compliant WRK directories
- [ ] work-queue-workflow SKILL.md updated with anatomy reference
- [ ] Related skills updated as applicable
