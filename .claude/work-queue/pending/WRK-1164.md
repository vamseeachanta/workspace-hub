## Mission
Build a power flow calculator for simple radial networks — bus injection model,
Newton-Raphson solver, and voltage/power results suitable for data center distribution analysis.

## What
- Module: `digitalmodel/src/digitalmodel/power/analysis/load_flow.py`
- Models: Bus (slack, PV, PQ), Branch (impedance, transformer), Network
- Solver: Newton-Raphson with Jacobian assembly for radial networks
- Output: bus voltages, branch flows, losses, convergence report
- Scope: radial/simple-loop networks (not full transmission-scale)

## Why
JD gap: "electrical topologies" — demonstrates ability to analyze power distribution
networks typical of data center and microgrid installations.

## Acceptance Criteria
- [ ] Bus and Branch dataclasses with per-unit conversion
- [ ] Y-bus (admittance matrix) assembly
- [ ] Newton-Raphson solver with configurable tolerance and max iterations
- [ ] Power mismatch and Jacobian calculation
- [ ] Results: voltage magnitude/angle, active/reactive flows, losses
- [ ] ≥18 TDD tests (3-bus, 5-bus, convergence, edge cases)
- [ ] All tests pass
