## Execution Notes (2026-03-04 — P1 Cross-Review Fixes)

Cross-review of WRK-1008 bundle (33KB, 7 files) produced the following actionable findings that were
addressed in the same session. Codex re-review returned **APPROVE** after all three P1 fixes applied.

**P1-A: `rg` alias not available in non-interactive bash (`classify_codex_failure`)**
- `rg` in the failure classifier and in `test-cross-review-path-sanitization.sh` relies on a Claude Code
  vendor alias that is only active in interactive shells.
- Fix: `rg -qi` -> `grep -Eqi` in `submit-to-codex.sh`; `rg -q` -> `grep -q`/`grep -qE` in test script.
- Commit: `cb07571f`

**P1-B: `run_renderer()` missing python3 fallback on uv failure**
- Original logic tried uv and returned 127 only if uv was ABSENT; never fell back to python3 if uv
  was present but the `uv run` command itself failed (e.g., transient import error, sandbox restriction).
- Fix: `uv run ... && return; if command -v python3; python3 ... && return; return 127`
- Verified: all 27/27 + 20/20 + 4/4 cross-review regression tests still pass after change.
- Commit: `cb07571f`

**P1-C: NO_OUTPUT hard-gate guard too permissive**
- `cross-review.sh` allowed CODEX_FALLBACK_ALLOWED=true even when `codex_exit` was non-zero from
  CLI failures, transport errors, or setup issues — only genuine model-empty responses (exits 0 or 5)
  should be eligible for 2-of-3 fallback consensus.
- Fix: added `[[ $codex_exit -eq 0 || $codex_exit -eq 5 ]]` guard in the NO_OUTPUT elif branch with
  inline comment explaining the semantics.
- Commit: `cb07571f`

**P1-D: MAJOR/MINOR verdict normalization (render-structured-review.py)**
- Codex sometimes returns `"verdict":"MAJOR"` or `"verdict":"MINOR"` instead of standard values.
  Renderer raised `ValueError: Invalid verdict: MAJOR`, causing exit 6 and raw fallback.
- Fix: added `_VERDICT_NORMALIZE` dict (`MAJOR->REQUEST_CHANGES`, `MINOR->APPROVE`, `NO CHANGES->APPROVE`,
  `NO_CHANGES->APPROVE`) applied before the `VALID_VERDICTS` guard in `_coerce_review()`.
- Commit: `edb5237d`
