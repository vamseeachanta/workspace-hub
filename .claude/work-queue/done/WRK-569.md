# docs: Map OrcaFlex .dat Model Library to Data-Source YAMLs

## What

Map the 5,496 OrcaFlex `.dat` model files from the document index (dde/Orcaflex)
into `specs/data-sources/` YAMLs for `digitalmodel`, `saipem`, and `rock-oil-field`.

| Project folder | Description | Cases | Domain | Standards |
