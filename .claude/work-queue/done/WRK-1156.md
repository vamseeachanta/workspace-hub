## Mission
Build a FAT/SAT/IST test procedure generator and results validator at
`digitalmodel/power/commissioning/test_sequence_validator.py`.

## What
- CommissioningPhase enum: FAT, SAT, IST
- TestStep: step_id, description, preconditions, action, acceptance criteria, dependencies
- GeneratorCommissioningSequence: build FAT/SAT/IST sequences from system config
- TestResultsValidator: load CSV results, validate against sequence, export punch list

## Why
Closes the FAT/SAT/IST commissioning gap. Demonstrates testability design, system verification
methodology, and technical documentation capability.

## Acceptance Criteria
- [ ] TestStep and CommissioningPhase data models — ≥5 tests
- [ ] GeneratorCommissioningSequence builds valid FAT/SAT/IST sequences — ≥10 tests
- [ ] TestResultsValidator loads results and validates — ≥8 tests
- [ ] Punch list export to CSV/markdown — ≥3 tests
- [ ] Module importable from `digitalmodel.power.commissioning`
