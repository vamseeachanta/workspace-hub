## Mission

Evaluate the quality and coverage of the canonical skill ecosystem after the WRK-639
cleanup (115 diverged + 9 incoming skills removed). Use skill-creator eval tooling
to score, benchmark, and improve skills systematically.

## What

1. **Eval pass**: Run skill-creator eval against all canonical skills — score each on
   triggering accuracy, content quality, and actionability
2. **Fix Phase 9 gaps**: Address the 18 skill coverage gaps flagged by nightly
   comprehensive-learning Phase 9 (missing back-links, missing script references)
3. **Verify WRK-639 cleanup**: Diff a sample of deleted "ahead" diverged copies against
   their canonical counterparts to confirm no valuable content was lost
4. **Improve bottom quartile**: For skills scoring below threshold, update content or
   descriptions using skill-creator optimize workflow

## Why

- WRK-639 deleted 124 skills based on file-level comparison without content evaluation
- Phase 9 flags 18 gaps nightly but nobody acts on them — they accumulate
- Skill triggering accuracy directly impacts agent productivity (wrong skill = wasted context)
- The skill-creator eval capability exists but has never been run ecosystem-wide

## Acceptance Criteria

- [ ] skill-creator eval run on all canonical skills with scores recorded
- [ ] 18 Phase 9 coverage gaps resolved (back-links added, scripts referenced)
- [ ] Sample of 10 deleted diverged "ahead" skills diffed against canonical — report on lost content
- [ ] Bottom quartile skills improved or flagged for removal
- [ ] Eval results saved to `specs/audit/skill-eval-<date>.yaml`
