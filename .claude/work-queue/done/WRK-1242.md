## Mission

Create a 16-section calculation methodology skill with per-section reference files that guides structured engineering analysis from scope through verification.

## Problem

Existing engineering calculations (geotechnical, CP, fatigue, pipeline stability) are well-implemented
but lack a unified **calculation methodology skill** that enforces the structured calculation approach:

1. **Problem definition** — scope, boundary conditions, design criteria
2. **Input gathering** — measured vs assumed, units, sources, validation ranges
3. **Methodology selection** — standard/code selection, applicability check, method comparison
4. **Computation** — step-by-step with intermediate results, hand-check points
5. **Validation** — sensitivity analysis, benchmark against known solutions, limit checks
6. **Reporting** — calculation-report skill integration, traceability to standard sections

The `calculation-report` skill handles rendering (step 6) but does not guide steps 1–5.

## Scope

- Create `.claude/skills/engineering/calculation-methodology/SKILL.md`
- Define the calculation methodology workflow as a structured checklist
- Integrate with existing `calculation-report` skill (step 6)
- Audit existing calculations (CP, geotechnical, fatigue) against the methodology
- Identify gaps where calculations skip validation or methodology selection steps

## Acceptance Criteria

- [x] Skill created with structured methodology stages
- [x] Checklist covers: problem definition, input validation, method selection, computation, validation, reporting
- [x] Audit of existing 15 calc report examples against methodology checklist
