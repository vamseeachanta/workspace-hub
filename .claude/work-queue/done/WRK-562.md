# docs(digitalmodel): Add Calc Examples Mapping — Marine Domain

## What

Add a `calc_examples` section to `specs/data-sources/digitalmodel.yaml` for the
**marine domain**, linking project wave scatter diagrams, RAO data, mooring fatigue
calculations, and buoy analysis workbooks to marine standard gaps
(DNV-OS-E301, API-RP-2P, API-RP-2SM).

## Why

Wave scatter diagrams, RAO tables, and mooring analysis workbooks represent the
practical test data for implementing marine dynamics modules. The long-term wave
scatter diagram (31245) and BOP fatigue calc (31290) are directly usable as
integration test fixtures.
