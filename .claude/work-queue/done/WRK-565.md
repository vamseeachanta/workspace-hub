# docs: Add Calc Examples Mapping — Installation / Umbilical Calcs

## What

Add `calc_examples` sections to `specs/data-sources/saipem.yaml` and
`specs/data-sources/rock-oil-field.yaml` for the **installation domain**,
linking project pipelay analysis, tensioner cart calcs, FDAS work scopes,
and TJG pipelay results to the relevant installation standards.

## Why

1,575 installation-domain calc files exist in the index (214 PDFs, 351 xls, 1,010 xlsx).
Many of these are project-specific pipelay, vessel motion, and umbilical analyses that
serve as reference implementations for portable installation analysis modules.

### Pipelay Analysis
- `0611-RES-0175-01 TJG Pipelay Analysis.xlsx`

### Tensioner / FDAS (Drilling Riser Installation)

## Steps

1. Read `specs/data-sources/saipem.yaml` and `specs/data-sources/rock-oil-field.yaml`
2. Query index for installation calc files: `query-docs.sh --domain installation`
3. Map pipelay calcs → saipem (S-lay, J-lay, umbilical installation)
4. Map tensioner/FDAS calcs → saipem (drilling riser tensioner)
5. Map general pipelay → rock-oil-field (OrcaFlex pipelay models)
6. Add `calc_examples:` sections to both YAMLs
7. Check if any installation standards are missing from these YAMLs
