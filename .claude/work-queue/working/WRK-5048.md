## Mission

Upgrade `scripts/work-queue/whats-next.sh` output to use consistent box-drawing tables and status emoji icons across all sections.

## What

Reformat the whats-next output to combine the best of three formats:
1. **Box-drawing tables** (┌─┬─┐ style) for all sections — consistent, professional look
2. **Status emoji icons** (✅ ARCHIVED, 🔄 WORKING, ⏳ PENDING, ★ HIGH, etc.) for quick visual scanning
3. **Grouped categories** (working, parked, unclaimed, high priority, newly unblocked, medium, blocked) — keep current grouping but all in uniform table format

## Why

Current output mixes plain-text column alignment (printf %-Ns) with ANSI color headers. The result is ragged when title lengths vary and hard to scan. A uniform box-drawing format with emoji status markers is more readable in terminal and in pasted output.

## Acceptance Criteria

- [ ] All sections use box-drawing table format (┌├└ etc.)
- [ ] Status emoji icons appear per-row: 🔄 working, ⏸ parked, ⚠ unclaimed, ★ high, ↑ unblocked, · medium, ✗ blocked, ◈ coordinating
- [ ] Notes/annotations (↳ lines) render cleanly inside or below the table
- [ ] Remote machine items still collapse to one-liner when local items exist
- [ ] Summary line and parallel hints section remain
- [ ] Column widths auto-adapt or are sensibly fixed for terminal widths ≥120
- [ ] Existing flags (--all, --category, --subcategory, --limit) still work
- [ ] TDD: test script output parsing for at least 3 sections
