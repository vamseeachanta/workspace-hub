# feat(workflow): enforce skill-driven /work gatepass from session start to close

## What

Create an enforceable, test-backed workflow so agents cannot bypass gatepass:
`/session-start` -> `/work` (work-queue orchestration) -> 9-stage verification evidence -> close/archive.

## Why

Current behavior shows drift between canonical process and skill docs, causing occasional plan/gate bypass. We need deterministic skill routing and machine-checkable checks across the full lifecycle.
