## Mission

Search for, download, and index naval architecture reference documents into the shared document-intelligence standards library at `/mnt/ace/docs/_standards/SNAME/` (or `Naval-Architecture/` subfolder).

## What

### Seed URLs (already known)

1. **Maritime.org Ship Plans Index** — https://maritime.org/doc/plans/index.php
2. **USNA Nimitz Library — Naval Architecture Guide (p=8554464)** — https://libguides.usna.edu/c.php?g=410506&p=8554464
3. **USNA Nimitz Library — Naval Architecture Guide (p=2795318)** — https://libguides.usna.edu/c.php?g=410506&p=2795318
4. **Principles of Naval Architecture Vol. 1 (SNAME)** — https://navalifpe.wordpress.com/wp-content/uploads/2011/09/principles-of-naval-architecture-vol-1-sname.pdf
5. **Reddit r/navalarchitecture — Study materials thread** — https://www.reddit.com/r/navalarchitecture/comments/1o8bb3o/where_can_i_get_study_materials_or_any_useful/
6. **Principles of Naval Architecture (Second Revision) Vol. I** — https://ftp.idu.ac.id/wp-content/uploads/ebook/tdg/ADNVANCED%20MILITARY%20PLATFORM%20DESIGN/Principles%20of%20Naval%20Architecture%20(Second%20Revision),%20Volume%20I.pdf
7. **Introduction to Naval Architecture — Tupper 1996** — https://home.hvl.no/ansatte/gste/ftp/MarinLab_files/Litteratur/IntroductionToNavalArchitecture_Tupper1996.pdf

### Execution steps

1. Fetch seed URLs — extract all linked PDFs and further resource links
2. Search online for additional freely-available naval architecture PDFs/textbooks (SNAME, IMO, MIT OpenCourseWare, etc.)
3. Download all available PDFs to `/mnt/ace/docs/_standards/SNAME/` (or create `Naval-Architecture/` subfolder)
4. Catalogue each download: title, author, year, source URL, local path
5. Add catalogue entries to `knowledge/seeds/naval-architecture-resources.yaml`
6. Regenerate document-index so files are searchable

## Why

Build a local offline library of naval architecture references to support digitalmodel engineering work and future module development.
