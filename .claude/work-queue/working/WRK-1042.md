# tidy-agent-teams: sentinel file support for non-conforming team names

## Mission
Allow `tidy-agent-teams.sh` to auto-delete teams with non-standard names by reading a
`.wrk-id` sentinel file inside the team directory, eliminating the need for manual deletion.

## What
- Add `.wrk-id` file support: if `~/.claude/teams/<team>/.wrk-id` contains a WRK-NNN ID
  and that WRK is archived, delete the team regardless of directory name
- Update `spawn-team.sh` to write `.wrk-id` when creating the team recipe
- Add tests for sentinel file detection

## Acceptance Criteria
- [ ] tidy reads `.wrk-id` sentinel when dir name does not match `wrk-NNN-slug`
- [ ] spawn-team.sh recipe includes `echo WRK-NNN > .../team-dir/.wrk-id`
- [ ] existing tests still pass
- [ ] new test: non-conforming dir with sentinel + archived WRK → deleted
