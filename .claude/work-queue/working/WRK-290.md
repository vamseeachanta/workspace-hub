# Install core engineering suite on ace-linux-2

## What

Install, configure, and verify the core open source engineering programs on ace-linux-2:
Blender, OpenFOAM, FreeCAD, Gmsh, and BemRosetta. Create a repeatable installation script
and verify each program launches and handles basic operations.

## Why

ace-linux-2 needs a complete engineering workstation setup. These five programs cover the
primary domains: 3D modeling, CFD, parametric CAD, mesh generation, and hydrodynamic BEM.

## Programs to Install

| Program | Version Strategy | Install Method | Verification |
|---------|-----------------|----------------|
--------------|
| **Blender** | Latest stable (4.x) | snap or apt PPA | Launch, import STL, render test |
| **OpenFOAM** | v2312+ or ESI latest | openfoam.org apt repo | `blockMesh` + `simpleFoam` on tutorial case |
| **FreeCAD** | 0.21+ or 1.0 RC | snap/flatpak/PPA | Launch, create Part, export STEP |
| **Gmsh** | 4.x latest | apt or download binary | Open, create geometry, mesh, export |
| **BemRosetta** | Latest | Build from source or AppImage | Launch, load test geometry |

### Additional supporting tools
| Program | Purpose | Install |
|---------|---------|
---------|
| **ParaView** | Post-processing / visualization | apt or download |
| **Python3 + pip** | Scripting across tools | Likely already present |
| **meshio** | Mesh format conversion (Python) | pip install |
| **PyFoam** | OpenFOAM Python utilities | pip install |

## Acceptance Criteria

- [ ] Blender installed and launches with GPU rendering (if available)
- [ ] OpenFOAM installed, `simpleFoam` runs cavity tutorial successfully
- [ ] FreeCAD installed, can create and export STEP file
- [ ] Gmsh installed, can mesh a simple geometry and export
- [ ] BemRosetta installed and launches
- [ ] ParaView installed for post-processing
- [ ] Supporting Python packages installed (meshio, PyFoam)
- [ ] Repeatable install script created at `scripts/setup/ace-linux-2-engineering-suite.sh`
- [ ] All programs verified with basic smoke test
