# FreeCAD Deep Parametric Engineering

## Mission

Enable production-grade FreeCAD automation for parametric hull generation with hydrostatic analysis, FEM preprocessing through the CalculiX solver chain, and design table driven parameter sweep studies with automated exports.

## What

The freecad-automation skill covers the API surface. This item adds depth for real engineering work:

1. **Parametric hull generation with hydrostatics** — define hull form via principal dimensions (L, B, T, Cb) → generate NURBS hull surface → compute hydrostatics (displacement, KM, BM, waterplane area) → export to STEP/STL for downstream solvers
2. **FEM preprocessing chain** — geometry → gmsh mesh → boundary conditions → material properties → CalculiX input deck → solve → extract stress/displacement results
3. **Design table driven studies** — YAML-defined parameter matrix (e.g., plate thickness, stiffener spacing, material grade) → batch FreeCAD model generation → batch FEM analysis → results comparison table
4. **Round-trip validation** — FreeCAD → STEP → gmsh mesh → verify watertightness → OpenFOAM/AQWA case setup (validates the full pipeline from CAD to solver)

## Why

FreeCAD is the only open-source parametric CAD tool in the stack. Making it work reliably for hull design and structural analysis enables rapid concept screening without commercial licenses. The FEM chain through CalculiX provides structural validation capability that currently requires ANSYS (licensed, limited to acma-ansys05).

## Acceptance Criteria

- [ ] Hull generator: input (L=100m, B=20m, T=8m, Cb=0.7) → FreeCAD NURBS hull → hydrostatics within 2% of analytical
- [ ] FEM chain: plate with hole → gmsh mesh → CalculiX → stress concentration factor within 5% of analytical (Kt ≈ 3.0)
- [ ] Design table: 3+ parameter variations → batch generation → results comparison YAML
- [ ] Round-trip: FreeCAD hull → STEP → gmsh → STL → verify closed surface (manifold check)
- [ ] All operations headless via freecadcmd (no GUI dependency)
- [ ] Integration tests with known engineering problems (simply supported beam, pressure vessel)
- [ ] Update freecad-automation skill with deep workflow sections
