## Mission

Assess and compare candidate methods for propeller-rudder interaction calculation, select the best fit for a fast-running Python model, and document the decision rationale.

## What

Using the literature from WRK-1148, evaluate each method family against:

| Criterion | Description |
|-----------|-------------|
| **Accuracy** | Agreement with experimental / RANS data across J range |
| **Input requirements** | What ship/propeller/rudder parameters are needed |
| **RPM / J range** | Valid from n=0 (engine-off) through full RPM drive-off |
| **Computational cost** | Suitable for fast-running (seconds, not hours) Python module |
| **Implementation complexity** | Can be coded without proprietary tools |

Candidate methods to compare:
- Actuator-disk + Rankine-Froude (simplest, least data)
- Holtrop-Mennen semi-empirical corrections
- Söding/Brix slipstream-in-rudder approach
- Molland & Turnock empirical lift/drag corrections
- Hybrid: momentum-based slipstream + empirical rudder coefficients

## Why

Multiple valid methods exist with different trade-offs. Choosing without explicit comparison risks implementing a method that fails outside its validity range (e.g. at low J during drive-off startup).

## Acceptance Criteria

- [ ] Comparison table: each method scored/ranked on the five criteria above
- [ ] Primary method selected with written rationale
- [ ] Fallback method identified for edge cases (engine-off free-wheeling, very low J)
- [ ] Known limitations and validation gaps documented
- [ ] Output: `docs/hydrodynamics/propeller-rudder-method-selection.md`
