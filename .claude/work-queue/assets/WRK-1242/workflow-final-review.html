Existing engineering calculations (geotechnical, CP, fatigue, pipeline stability) are well-implemented but lack a unified calculation methodology skill that enforces the structured calculation approach:

  1. Problem definition — scope, boundary conditions, design criteria
  2. Input gathering — measured vs assumed, units, sources, validation ranges
  3. Methodology selection — standard/code selection, applicability check, method comparison
  4. Computation — step-by-step with intermediate results, hand-check points
  5. Validation — sensitivity analysis, benchmark against known solutions, limit checks
  6. Reporting — calculation-report skill integration, traceability to standard sections

The calculation-report skill handles rendering (step 6) but does not guide steps 1–5.
