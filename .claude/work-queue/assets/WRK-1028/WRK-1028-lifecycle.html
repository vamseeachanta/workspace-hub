Acceptance Criteria
  • ☐ 20 per-stage YAML contracts at `scripts/work-queue/stages/stage-NN-<slug>.yaml`
  • ☐ Each contract: `order`, `name`, `weight`, `invocation` (task_agent|human_session|chained_agent), `human_gate`, `skills_required[]`, `entry_reads[]`, `exit_artifacts[]`, `blocking_condition`, `log_action`, `context_budget_kb`
  • ☐ `start-stage.sh WRK-NNN <N>`: reads contract → if `invocation=task_agent` spawns Task agent with constructed prompt (contract + entry_reads + micro-skill); if `invocation=human_session` loads artifacts into current session and emits checklist
  • ☐ `exit-stage.sh WRK-NNN <N>`: validates `exit_artifacts[]`, checks human_gate artifact if `human_gate=true`, writes stage log (`log_action`), updates `stage-state.yaml`, commits, prints Stage N+1 summary

  • ☐ 20 micro-skills at `.claude/skills/workspace-hub/stages/stage-NN-<slug>.md` (≤20 lines each)
  • ☐ Each micro-skill: purpose (1 line), entry artifacts to read, checklist (≤8 items), exit conditions, sub-skills to invoke for heavy stages
