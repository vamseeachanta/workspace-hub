Prompt Start Context

Build a living knowledge layer on top of all documents stored across the two machines (/mnt/local-analysis, /mnt/ace, /mnt/dde) so that their contents can be surfaced, cross-referenced, and systematically used to enhance engineering work across all repos.

This is a phased, compound work item. Each phase gates the next. Phase G spawns the mission-driven WRK items for each repo once the enhancement plan is approved.

Why

Thousands of standards, reports, papers, and reference documents exist on the drives. Currently they are opaque to agents. Indexing and linking them turns static files into active context that can inform code, calculations, and decisions — without manual retrieval.
