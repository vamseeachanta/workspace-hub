Prompt Start Context

The agentic AI landscape is on a steep trajectory. By mid-2026: - Agents will run longer, more reliably, with less context loss - Multi-agent coordination will be native, not orchestrated manually - Model quality for code, reasoning, and planning will be significantly higher - MCP and plugin ecosystems will be mature enough to depend on - Autonomous background work (research, curation, gap surfacing) will be reliable

Every repo in the ecosystem either positions us to exploit that trajectory or will become a maintenance burden. This roadmap makes the call on each.
