Prompt Start Context

Perform BSEE data analysis for the Buckskin field, which covers six lease blocks in the Keathley Canyon: - KC 785 - KC 828 - KC 829 - KC 830 - KC 871 - KC 872

Extract and analyze all available BSEE data for these blocks including production data, wellbore information, drilling/completion activities, and intervention history. Present results under the field nickname "Buckskin."

Why

Buckskin is a specific field of interest requiring a focused data package. This serves as both a deliverable and a validation case for the broader BSEE field data pipeline (WRK-017).

Acceptance Criteria

  • [x] Query BSEE data for all six Keathley Canyon lease blocks (785, 828, 829, 830, 871, 872)
  • [x] Map BOEM OCS lease numbers (G 25806, G 25813, G 25814, G 25815, G 25823)
  • [x] Add Buckskin to database tables in Part 1 and Part 2
  • [x] Production data summary (oil, gas, water) by block and aggregated for field
  • [x] Validate against known benchmarks: ~30,000 bopd initial, top-10 wells in 2021/2022
  • [x] Wellbore inventory: count, types, status per block (discovery well, 3 appraisal, production wells)
  • [x] Drilling and completion activity timeline (2009 discovery through current)
  • [x] Intervention activity summary
  • [x] Note Buckskin South Expansion (Dec 2025 contract)
  • [x] Results labeled with field nickname "Buckskin"
  • [x] Visualizations: production trends, well count over time, phased development timeline
  • [x] Cross-reference with lower tertiary / Wilcox classification
  • [x] Document operator transitions: Repsol → LLOG → Harbour Energy (2026)
