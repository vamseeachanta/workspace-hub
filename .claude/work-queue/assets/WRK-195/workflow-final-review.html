Why

BSEE covers the GoM OCS at well/lease level — but the US produces ~13 million bopd total (2025) and GoM is only ~1.8 million bopd (~14%). The other 86%: Permian Basin (~6.6M bopd, ~50% of US total), Bakken, Eagle Ford, DJ Basin, Alaska (~0.4M bopd, growing) — is covered exclusively by EIA, not BSEE.

EIA provides the best-documented open API in the global O&G data landscape: free REST API, consistent series IDs, state and basin-level monthly data, and an international country-level database that covers 200+ countries. Adding EIA completes the US picture and provides the global aggregate layer for cross-basin analysis (GoM vs Permian vs Alaska vs international).

Acceptance Criteria

  • [ ] EIA API client (v2) with API key, rate limiting, local cache, series ID resolution
  • [ ] State-level monthly crude production for all 50 states (2000 to present)
  • [ ] Basin-level Drilling Productivity Report loader (Permian, Bakken, Eagle Ford, etc.)
  • [ ] Alaska field-level breakdown (Prudhoe Bay, Kuparuk, Point Thomson)
  • [ ] International country-level monthly production (160+ countries)
  • [ ] Hyperbolic DCA for shale basins (b-factor > 1 vs exponential for conventional)
  • [ ] GoM crosscheck: compare EIA GoM series vs BSEE aggregates for validation
  • [ ] 30+ tests
  • [ ] Cross-basin comparison example: Permian vs GoM vs NCS vs Brazil NPV economics
