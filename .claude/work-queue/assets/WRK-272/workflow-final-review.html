Prompt Start Context

Extend the CathodicProtection class with two new calculation routes:

  1. DNV-RP-B401 offshore fixed platform SACP — the primary offshore engineering use case not yet implemented. DNV-RP-B401 (currently 2011 in-repo; 2021 edition to be confirmed by WRK-269) covers jacket structures, gravity-based structures, and subsea equipment.

  2. DNV-RP-F103 2016 update — if WRK-269 confirms material differences from the 2010 edition currently implemented. The Saipem Yellowtail project used the 2016 edition; key areas to check: wet storage period handling, stand-off anode calculations, FLET/PLET protection clauses.

Blocked by WRK-269 — scope of this item depends on what the standards research confirms. WRK-269 will determine: - Whether DNV-RP-B401 2021 significantly changes the 2011 calculation method - Whether DNV-RP-F103 2016 changes are material or cosmetic relative to 2010 - Which editions we can validate against

Why

Current code covers submarine pipelines (DNV-RP-F103) and ships (ABS). Offshore fixed platforms are the highest-value A&CE use case — jacket structures, jackets with anodes, topsides — and have no implementation. This is the core of offshore CP engineering.

WRK-206 (archived) noted the gap: "current engineering skills cover cathodic-protection (prevention)" but flagged that the skill and FFS are the prevention/damage pair. The offshore platform route completes the prevention side.
