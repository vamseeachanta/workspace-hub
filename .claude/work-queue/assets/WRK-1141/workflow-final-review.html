1. scripts/hooks/post-commit — auto-push after every commit

Mirror the existing post-merge / post-checkout / post-rewrite hook pattern.

#!/usr/bin/env bash
# post-commit: auto-push to remote after every commit (WRK-1141)
# Guards: skip if no remote, detached HEAD, CI environment, or dry-run flag

Guards required: - SKIP_PUSH=1 env var → skip (lets scripts opt out) - CI=true / GITHUB_ACTIONS → skip - Detached HEAD → skip - No upstream remote configured → skip with warning - Submodule context → skip (hub-level push only)

2. scripts/work-queue/start-wrk.sh WRK-NNN — trunk vs branch routing

Reads WRK frontmatter (complexity, compound, estimated session count) and: - simple / single-session → prints "Commit to main directly" - medium / multi-session → git checkout -b <prefix>/WRK-NNN-<slug> - complex / compound → same as medium, with worktree option

Removes the human decision point from session start.

3. Wire into scripts/setup/install-all-hooks.sh

Add post-commit to the canonical hooks install list so it propagates to all machines via post-merge re-install.

Steps

  1. Write scripts/hooks/post-commit

Guards (exit 0 silently unless noted): - SKIP_PUSH=1 → skip - CI=true / GITHUB_ACTIONS → skip - Detached HEAD (git symbolic-ref HEAD fails) → skip - No upstream configured (git rev-parse @{u} fails) → warn + skip - Submodule context (git rev-parse --show-superproject-working-tree non-empty) → skip - Rebase in progress (.git/rebase-merge or .git/rebase-apply exists) → skip - Cherry-pick in progress (.git/CHERRY_PICK_HEAD exists) → skip - Amend/reword (GIT_REFLOG_ACTION contains "amend" or "reword") → skip - Merge conflict (.git/MERGE_HEAD exists) → skip

On pass: background push with log: bash (git push origin HEAD > /tmp/post-commit-push.log 2>&1 &)

  1. Write scripts/work-queue/start-wrk.sh WRK-NNN

Routing precedence: 1. compound: true → feature branch regardless of complexity 2. complexity: simple → print "Commit to main directly" 3. complexity: medium or complexity: complex → git checkout -b feature/WRK-NNN-<slug>

Branch-already-exists: print warning, exit 0 (let user handle) Frontmatter parsing: grep -m1 "^field:" file.md | sed 's/^field: *//' Slug: lowercase title, first 5 words, hyphenated

  1. Update scripts/setup/verify-setup.sh

  2. Add post-commit to the verified hooks list

  4. SKIP_PUSH=1 guard

  7. No upstream guard
  8. Submodule guard
  9. Rebase-in-progress guard
  10. Amend guard (GIT_REFLOG_ACTION=amend)
  11. Normal path: push fires in background

Write tests/scripts/test_start_wrk.sh — ≥5 tests: - simple → main message - medium → branch creation - complex → branch creation - compound=true → branch regardless of complexity - Branch-already-exists → warning, exit 0

  1. Commit + push
