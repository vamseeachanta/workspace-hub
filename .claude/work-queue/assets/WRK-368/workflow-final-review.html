Create .claude/skills/workspace-hub/repo-structure/SKILL.md as the authoritative reference for source code layout, test mirroring, root directory cleanliness, gitignore compliance, and agent infrastructure rules across all workspace-hub tier-1 repos.

Why

Agents need a single invocable reference (/repo-structure) to self-check compliance before and after making structural changes. This complements /file-taxonomy (output file placement) with source layout rules.
