Title
feat(hydrodynamics): integrate wavespectra library for spectral processing
Status
pending
Route
-
Orchestrator

Prompt Start Context

WRK-383 identified hydrodynamics/wave_spectra as a gap module. While basic JONSWAP/PM implementations exist, the wavespectra library provides a complete, industry-standard solution including 15+ format readers (ERA5, WW3, SWAN, NDBC) and 60+ spectral parameter methods.

Why

Integrate the wavespectra library (v4.4.2) into digitalmodel/hydrodynamics to close the gap identified in WRK-383 and enhanced by WRK-391.

Acceptance Criteria

  • wavespectra added to project dependencies.
  • digitalmodel.hydrodynamics.wave_spectra exposes wavespectra capabilities.
  • Unit tests confirm correct spectral fitting and partitioning.
  • No regression in existing WaveSpectra tests.

Plan

  1. Add wavespectra as a dependency in digitalmodel.
  2. Implement a thin wrapper in digitalmodel.hydrodynamics.wave_spectra that utilizes wavespectra.
  3. Provide factory methods to fit JONSWAP/PM spectra using the library's fitting routines.
  4. Implement partitioning methods leveraging wavespectra's capabilities.
  5. Add unit tests verifying integration and correct parameter extraction.
