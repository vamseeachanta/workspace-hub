  • Detects which repos have staged changes (via git diff --name-only origin/HEAD..HEAD)
  • Runs scripts/testing/run-all-tests.sh --repo <name> for each changed repo
  • Runs scripts/quality/check-all.sh --repo <name> for each changed repo
  • Blocks push if any unexpected test failures or ruff errors (exit code ≠ 0)
  • Skip with --no-verify documented but discouraged (flag use logged)
  • Install script: scripts/hooks/install-hooks.sh — symlinks pre-push into .git/hooks/ for hub and each submodule
  • Fast mode: --changed-only flag (default) runs only affected repos; --all runs full suite

  6. Detect changed tier-1 repos: git diff --name-only <remote_oid>..<local_oid>
  7. --all or run_all=true: run all 5 repos regardless
  8. For each repo: check-all.sh --repo <name> then run-all-tests.sh --repo <name>
