Phase 3 — Session-exit tidy - Extend the existing Stop hook (emit-session-quality-signals.sh or a new tidy-agent-teams.sh) to: 1. Detect any teams whose WRK is now archived/closed → TeamDelete them 2. Detect task UUID dirs older than 7 days with no active team reference → purge 3. Emit a one-line summary signal: agent_teams_tidied: N_deleted, M_tasks_purged - Keep the hook fast (<5s): use shell+jq only, no Python
