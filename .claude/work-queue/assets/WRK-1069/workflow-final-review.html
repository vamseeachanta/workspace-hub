Phase 2 — wrk-cost-report.py

wrk_cost_report(wrk_id=None) → table
  load cost-tracking.jsonl
  filter by wrk_id if provided
  group by wrk_id → sum(input_tokens), sum(output_tokens), sum(cost_usd)
  join with WRK frontmatter (complexity, subcategory, status)
  sort: by cost_usd desc
  output: table [wrk_id | complexity | subcategory | input_tok | output_tok | cost_usd]

Step 2.1 — Write scripts/ai/wrk-cost-report.py: - CLI: wrk-cost-report.py [WRK-NNN] (optional WRK filter) - Reads config/ai-tools/pricing.yaml for rate verification (compare against stored cost_usd) - Reads .claude/state/session-signals/cost-tracking.jsonl - Output: aligned table + grand total row - Exit 0 on success, 1 if no data found

Step 2.2 — Handle edge cases: missing wrk field (attribute to "unattributed"), missing cost_usd (recalculate from tokens + pricing.yaml), empty file.

Exit: wrk-cost-report.py WRK-1069 runs without error; wrk-cost-report.py prints all-WRK summary table.
