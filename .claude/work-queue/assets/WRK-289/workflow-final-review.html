Title
Research open source FEA programs for engineering assignments
Status
done

Prompt Start Context

Conduct a thorough research survey of open source Finite Element Analysis (FEA) programs available for Linux. Evaluate each for suitability in engineering assignments — structural, thermal, and multiphysics analysis. Produce a recommendation report with installation notes.

Why

Before installing FEA tools on ace-linux-2, we need to know which programs are actively maintained, well-documented, and practical for real engineering coursework/assignments. Bad tool choices waste setup time and create workflow dead-ends.

Acceptance Criteria

  • [ ] At least 8 FEA programs evaluated against all criteria
  • [ ] Comparison matrix completed
  • [ ] Top 3 recommendations identified with clear rationale
  • [ ] Installation notes for each recommendation (Linux/Ubuntu)
  • [ ] At least 2 example assignment workflows described
  • [ ] Report written to docs/research/open-source-fea-survey.md
