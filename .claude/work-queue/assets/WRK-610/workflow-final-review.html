Prompt Start Context

Restructure and integrate CAD-DEVELOPMENTS (a collaborator repo at github.com/bakkiprasad5669/CAD-DEVELOPMENTS) as a workspace-hub submodule with the right directory structure, README, and toolchain config to serve its mission: AI-agent-driven FEA model development from scratch (geometry → mesh → solve → post-process pipeline executed by AI agents on ace-linux-2).

Current state: 13 flat files (12 FreeCAD .FCStd ship/boat hull models + 1 Blender CTD.blend) with a one-line README, sitting untracked at workspace-hub root.

Why

The repo has a high-leverage mission (AI agents that build FEA models end-to-end) but no structure to support it. Without a proper pipeline layout, agent handoffs, and ecosystem registration, the work will stay isolated and ad hoc. Establishing the foundation now enables WRK-394 (planing hull dynamics) and future structural FEA/CFD work to reuse this infrastructure.

  hull/         ← CalculiX .inp / FEniCS .py per analysis case

agents/         ← Python agent stubs for each pipeline stage
