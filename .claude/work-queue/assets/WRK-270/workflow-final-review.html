Prompt Start Context

The cathodic-protection SKILL.md (v1.0.0) contains Python examples referencing classes that do not exist in the codebase: AnodeDesign, PipelineCP, CoatingAnalysis, ICCPDesign, CPMonitoring, SurveyAnalysis, InterferenceAnalysis. Anyone following these examples will get ImportError.

Why

A skill with broken examples actively misleads — an agent following the skill will write code that fails immediately. This is worse than no skill.
