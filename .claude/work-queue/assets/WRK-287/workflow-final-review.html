Prompt Start Context

Configure network file sharing so that ace-linux-2 (another machine on the same LAN) can access /mnt/local-analysis/workspace-hub from ace-linux-1 (192.168.1.100) seamlessly — as if it were a local directory. Windows sharing is out of scope for now.
