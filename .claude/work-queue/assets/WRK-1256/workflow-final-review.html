Step 1: Deterministic watermark/dedup filter

Add to table_exporter.py: - Content hash dedup: hash each table's content; skip if seen before in same doc - Watermark pattern list: regex for known watermarks (IHS Licensee, DNV copyright, etc.) - Min-content gate: skip tables with <3 data rows OR <2 columns with numeric values

Step 2: LLM quality rating (Haiku)

After deterministic filters, rate surviving tables via Haiku: - Input: first 10 rows of CSV as text - Prompt: "Rate this extracted table: usable (clean engineering data) / partial (some useful data, needs cleanup) / junk (headers only, formatting artifacts, no data)" - Output: quality field in extraction report YAML - Cost: ~$0.0001/table × ~400 real tables = ~$0.04 total for existing corpus - For batch runs: rate a sample (every 10th table) to estimate quality without rating all

Step 3: Retroactive quality audit

Re-score all 664 existing tables:

uv run --no-project python scripts/data/doc-intelligence/rate-table-quality.py \
    --input data/doc-intelligence/deep/tables/ --output data/doc-intelligence/table-quality-audit.yaml

Step 4: Quality report

Generate per-document and per-domain quality summary: - Tables: total, real, watermark, empty, usable, partial, junk - Quality rate = usable / total - Flag documents with <50% quality rate for parser investigation
