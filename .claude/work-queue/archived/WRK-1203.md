- `tests/asset_integrity/` — `import imp` (Python 3.12 incompatible)
- `tests/cross_repo/` — compliance test failures

- `tests/compat/` — hardcoded count assertions

### Also fix:

- pytest capture bug with subprocess.run (upgrade pytest or pin version)
- `filterwarnings = error` + `ignore::DeprecationWarning` interaction

## Why
Quality Gates CI should pass clean for confidence in main branch stability.
