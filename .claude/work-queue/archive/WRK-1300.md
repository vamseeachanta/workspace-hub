### Session Learnings to Capture
- pdfplumber hangs in kernel D-state on NTFS and NFS — uninterruptible by SIGALRM
- pdftotext (poppler) is 37x faster and subprocess timeout is killable
- Multiprocessing with subprocess.run(timeout=N) is the reliable pattern
- Path remapping patterns for cross-machine file access
- Index copy-process-copy-back pattern for cross-machine batch work
