## Mission
Build a rule-based microgrid energy management controller at
`digitalmodel/power/microgrid/microgrid_controller.py` for grid-connected and island-mode operation.

## What
- MicrogridMode enum: GRID_CONNECTED, ISLANDED, TRANSITIONING, BLACK_START
- DERAsset: PV/BESS/GENSET/FUEL_CELL with dispatch priority and rated power
- BESSController: SOC management, charge/discharge rate computation
- IslandDetector: ROCOF computation, vector shift detection, anti-islanding trip logic
- MicrogridEMS: merit-order dispatch, island/grid transition sequencing

## Why
Differentiating module — BESS/islanding knowledge elevates candidate above O&G-only engineers.
Demonstrates microgrid controller design, DER integration, IEEE 1547.4 islanding detection,
and seamless transfer logic.

## Acceptance Criteria
- [x] MicrogridMode + DERAsset data models — 8 tests (≥5 required)
- [x] BESSController with SOC boundaries and rate logic — 14 tests (≥8 required)
- [x] IslandDetector with ROCOF and vector shift — 10 tests (≥8 required)
- [x] MicrogridEMS with merit-order dispatch and mode transitions — 10 tests (≥10 required)
- [x] All classes have docstrings referencing IEEE 1547.4
- [x] Module importable from `digitalmodel.power.microgrid`
