## Mission
Give agents a fast, structured way to find "where is X done" across all five
tier-1 Python repos without reading entire directory trees — reducing codebase
orientation time from minutes to seconds.

## What
- Script: `scripts/search/cross-repo-search.sh <pattern> [--type py|md|yaml] [--repo <name>]`
  - Wraps ripgrep with correct per-repo PYTHONPATH and src/ root awareness
  - Outputs: file:line:match with repo prefix for unambiguous location
  - Ranks results: src/ hits before tests/, tests/ before docs/
- Symbol index: `scripts/search/build-symbol-index.py`
  - Walks all tier-1 repos, extracts Python class/function definitions via `ast`
  - Writes `config/search/symbol-index.jsonl` — one entry per symbol:
    `{"symbol": "SaveData", "kind": "class", "repo": "assetutilities", "file": "...", "line": N}`
  - Fast lookup: `scripts/search/find-symbol.sh <name>` reads the index, no AST parsing at query time
- Rebuild trigger: post-merge git hook updates index when `src/` files change
- Integration: `/work status WRK-NNN` can cross-reference target_repos symbols

## Why
The 112 engineering WRKs frequently ask "where is the CP anode design calculation?"
or "which module handles yfinance calls?". Agents currently run multiple Glob/Grep
rounds to answer this. A pre-built symbol index answers in one lookup.

## Plan

**Route B — Medium complexity**

### Phase 1: Symbol Indexer (3pts)
1. `scripts/search/build-symbol-index.py` — AST walker over all 5 tier-1 `src/` roots
   - Emits `class`, `function`, `method`, `constant` records to `config/search/symbol-index.jsonl`
   - Graceful skip + warning on parse errors (R3 mitigation)
2. `config/search/symbol-index.jsonl` — pre-built index (22,697 symbols, 2,644 files)

### Phase 2: Fast Lookup (1pt)
3. `scripts/search/find-symbol.sh <name> [--kind] [--repo]` — jq path with grep+python fallback
   - Exits 1 on miss; <1s on existing index

### Phase 3: Cross-Repo Search (2pts)
4. `scripts/search/cross-repo-search.sh <pattern> [--type] [--repo]` — rg preferred, grep -r fallback
   - Ranked output: `src/` → `tests/` → `docs/`

### Phase 4: Hook Integration (1pt)
5. `scripts/hooks/post-merge` — conditional index rebuild when `src/` files change

### Phase 5: Tests (2pts)
6. `tests/search/test-symbol-index.sh` — 8 test cases: happy path, miss, ranking, broken-file skip, filters

### Test Strategy
