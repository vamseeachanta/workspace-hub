## Mission

Create a script that parses session logs into structured skill execution records for automated skill improvement.

## What

Build `scripts/skills/skill-execution-tracker.py` that:
1. Reads session logs from `.claude/state/sessions/`
2. Extracts skill invocations with outcomes (skill_name, task, success/failure, error, timestamp)
3. Writes to `skill-executions.jsonl` (append-only)
4. Integrates with comprehensive-learning nightly pipeline

## Why

From WRK-5085 research (cognee pattern): structured observation of skill executions enables evidence-based improvement. Currently session logs are unstructured — this creates the data layer for automated skill improvement.

## Acceptance Criteria

- [x] Script parses session logs and extracts skill invocations
- [x] Output schema: {skill_name, timestamp, session_id, project, duration_s}
- [x] Append-only JSONL output (grep-friendly per coding style rules)
- [x] TDD tests — 13 passing (parse, extract, write, edge cases)
