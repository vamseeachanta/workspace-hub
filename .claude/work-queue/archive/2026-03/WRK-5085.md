## Research Findings (2026-03-15)

### Source 1: karpathy/autoresearch
- **Pattern**: Agent edits one file → fixed-budget eval → keep if better, revert if not → git as state machine
- **Fit**: HIGH — replace `train.py` with `SKILL.md`, metric with skill-eval score
- **Effort**: 1-2 days

### Source 2: Cognee self-improving skills (tricalt thread)
- **Pattern**: 5-stage loop: Ingest → Observe → Inspect → Amend → Evaluate
- **Key insight**: Track (skill, task, outcome, error) tuples to find which skills fail on which patterns
- **Fit**: MEDIUM — patterns right, implementation heavy (graph DB overkill; JSONL sufficient)
