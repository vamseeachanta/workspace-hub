## Mission

Fix 111 broken `related_skills` cross-references in SKILL.md frontmatter identified by
WRK-1244 eval. Create a deterministic script to resolve or remove stale references.

## What

1. Create `scripts/skills/fix-related-skills.py` that:
   - Builds name index of all current skills
   - For each `related_skills` reference, fuzzy-matches to canonical name
   - Fixes resolvable references, removes truly dead ones
   - Dry-run by default, `--apply` to write
2. Run with `--apply` and verify via eval-skills.py re-run

## Why

- 111 warnings from unresolved cross-references reduce eval signal-to-noise
- Broken references mislead agents into looking for non-existent skills

## Acceptance Criteria

- [x] Script created at `scripts/skills/fix_related_skills.py`
- [x] `related_skill_unresolved` warnings reduced to 0 (was 111)
- [x] eval-skills.py re-run shows improvement — warning eliminated entirely
