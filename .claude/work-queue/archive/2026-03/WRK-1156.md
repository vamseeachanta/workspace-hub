# chore(harness): Stage-07/17 Gate-Passed Printer Script

## Mission

Create a deterministic script that prints the canonical gate-passed message
and checkpoint prompt when a hard-gate stage is approved, replacing the
LLM prose step with a reproducible, auditable output.

## What

Script: `scripts/work-queue/print-gate-passed.sh WRK-NNN STAGE`

- Validate that `evidence/plan-final-review.yaml` (stage 7) or
  `evidence/user-review-close.yaml` (stage 17) has `decision: passed/approved`
- If valid: print `GATE PASSED — /checkpoint WRK-NNN → new session → /resume WRK-NNN`
- Exit 0 on pass; exit 1 on not-yet-approved

## Acceptance Criteria

- [ ] Script prints gate-passed message when evidence decision is approved/passed
- [ ] Script exits 1 with clear message when gate not yet approved
- [ ] Works for both stage 7 and stage 17 (auto-detects from evidence)
