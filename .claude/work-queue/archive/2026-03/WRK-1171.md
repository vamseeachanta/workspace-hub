## Mission

Create a data-driven radar chart visualization of AI agent capabilities with automated daily regeneration.

## What

Build an interactive HTML radar chart comparing workspace-hub AI providers (Claude, Codex, Gemini, OpenAI) and Claude model tiers across capability dimensions, driven from YAML config and regenerated daily via cron.

## Why

The frontier model comparison chart (provided by user) demonstrated the value of visual capability profiles. Applying this pattern to our own agent ecosystem gives immediate routing clarity — which provider for which task, which Claude tier for which route — without parsing multiple YAML configs.

- [x] AC1: Radar chart HTML with two panels — provider comparison + Claude tier routing
