## Mission
Check that all public API symbols across tier-1 repos have corresponding
reference documentation entries, and that docs/ structure meets baseline standards.

## What
- Public symbol audit: enumerate `__all__` (or all public classes/functions)
  per repo; flag any missing docstrings or docs/ entries
- docs/ structure check: expected files (index, changelog) present
- Sphinx/mkdocs detection: report build system in use per repo
- Output: per-repo API coverage % + docs structure report

## Why
API reference gaps make it hard for new contributors to use the libraries
and create maintenance debt as the codebase grows.

## Acceptance Criteria
- [ ] Public symbol audit runs across all 5 tier-1 repos
- [ ] Per-repo API docstring coverage % reported
- [ ] docs/ structure validated; missing files listed
- [ ] Codex cross-review passes
