# Standardize mount paths — both machines

## Summary

Replaced unreliable desktop auto-mount paths (spaces, cryptic labels, not in fstab) with
clean, UUID-based fstab entries on both Linux workstations. All data drives now mount
automatically on headless boot and SSH-only sessions.

## Key Decision

**Do NOT rename `/mnt/ace` on ace-linux-1.** Blast radius analysis found 378,104 files in
`data/`, 26 scripts, 9 skills, 8 specs, 2 docs, 2 configs, and ace-linux-2's SSHFS fstab
entry all reference `/mnt/ace`. Cost of rename far exceeds benefit. Descriptive symlink
`/mnt/ace-data` added instead.
