## Implementation Notes (2026-02-16)

### Finding: No Stub Files
The original WRK description referenced "15 stub Excel files" but the actual BSEE data uses **text files inside ZIP archives**, not Excel. All 15 INC files + 2 investigation files contain real data — none are stubs. The smallest file (`mv_incs_shutins_alaska.txt`, 718 bytes) is real data with few records.

### Civil Penalties
BSEE public data tracks Incidents of Non-Compliance (INCs) as **counts** (WARNING_INCS, COMP_SHUTIN_INCS, FAC_SHUTIN_INCS) rather than dollar amounts. The `ViolationIncident` model has a `penalty_amount` field, but BSEE doesn't publish penalty dollar amounts in their public bulk data downloads. Civil penalty dollar amounts would need to be sourced from BSEE enforcement action press releases or FOIA requests.

### No Code Changes Required
All data files were refreshed from live BSEE endpoints. No code modifications needed — the acquirer and importers work correctly. The only issue is the statistics URL which has always been broken (documented via TODO in the code).
