# NCS Production Data Module — NPD/Sodir Open Data Integration

## Motivation

BSEE provides GoM field-level production, well, and facility data — and worldenergydata already has a full module for it (WRK-019, WRK-077, WRK-111). The Norwegian Continental Shelf (NCS) has an equivalent open data source: the Norwegian Petroleum Directorate (NPD) / Sodir factpages API. Adding NCS data gives:

- A second basin for cross-basin field development economics (NCS vs GoM)
- Real production data to anchor the Aker BP Solveig Phase 2 case study (WRK-080 Phase 2)
- Decline curve validation data for the production engineering module (WRK-164)
- Broader scope for the NPV field development calculator (WRK-081)

## What

Build a `worldenergydata/ncs/` module mirroring the structure of `worldenergydata/bsee/`:

### Data Sources

**NPD Factpages** (open, no auth):
- `https://factpages.npd.no/en/field/pageview/all` — field browser
- REST/OData API endpoints for programmatic access:
  - `https://factpages.npd.no/ReportServer?...` (legacy)
  - `https://api.npd.no/...` (newer JSON API — verify availability)
- Monthly field production: oil (Sm³), gas (Sm³), NGL (tonnes), condensate, water injected
- Well data: completion, perforation, test records
- Discovery data: year, resource estimate, development status
- Facility data: installation type, operator, coordinates

### Fields of Interest (priority order)

1. **Solveig** — Aker BP (65%), subsea tieback to Edvard Grieg; 39 MMboe; prod start 2026-02-02
2. **Edvard Grieg** — Aker BP, host platform; production history since 2015
3. **Ivar Aasen** — Aker BP, nearby; subsea tieback comparison
4. **Valhall** — Aker BP, mature field; good for decline curve study
5. **Johan Sverdrup** — Equinor, large platform development; NPV benchmark

## Module Structure

```
worldenergydata/
  ncs/
    __init__.py
    production/
      __init__.py
      field_production.py       # Monthly production loader (mirrors bsee/production/)
      decline_curves.py         # DCA for NCS fields (mirrors bsee/analysis/forecasting/)
    facilities/
      __init__.py
      facility_loader.py        # Installation/platform data
    wells/
      __init__.py
      well_loader.py            # NCS well data
    analysis/
      __init__.py
      field_economics.py        # NPV/IRR using NCS production + cost benchmarks
    data/                       # Cached NPD data (gitignored if large)
```

## Acceptance Criteria

- [ ] NPD API client with rate limiting and caching (mirrors BSEE loader pattern)
- [ ] Monthly field production loader: oil, gas, NGL, water injection per field
- [ ] LFS stub handling if data files are large (consistent with BSEE pattern)
- [ ] Decline curve fitting for at least 3 NCS fields (Edvard Grieg, Valhall, Ivar Aasen)
- [ ] Field economics example: Solveig NPV with NPD production profile as input
- [ ] 30+ tests (unit + integration)
- [ ] Data dictionary: units, field name conventions, NPD field codes vs. common names

## Plan Notes

### Phase 1 — API Exploration & Client (~2 hours)
- Audit NPD API endpoints; confirm JSON vs. OData vs. CSV download availability
- Build `NpdClient` with caching to avoid repeated downloads (same pattern as BSEE loaders)
- Handle NCS unit conventions: Sm³/d for oil, MSm³/d for gas, convert to bbl/d for cross-basin comparison

### Phase 2 — Production Loader (~3 hours)
- `FieldProductionLoader`: load monthly production per field by NPD field code or name
- Normalise column names to match BSEE production schema where possible (for cross-basin analysis)
- Graceful handling of missing months, incomplete records (same LFS-stub pattern as BSEE)

### Phase 3 — Decline Curves & Economics (~3 hours)
- Reuse `worldenergydata/bsee/analysis/forecasting/decline_curves.py` pattern
- NCS-specific parameters: Norwegian royalty/tax regime (78% marginal rate, uplift allowance)
- Solveig example: fit decline curve to Edvard Grieg production (available since 2015), project Solveig plateau extension contribution

### Phase 4 — Tests
- Mock NPD API responses for unit tests (no live API in CI)
- Integration test: fetch Edvard Grieg production and run decline curve fit
- Cross-validation: compare NPD field total with published Aker BP annual reports
