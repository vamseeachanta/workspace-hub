# Marine Safety Importer Validation

## What
Validate existing marine safety importers (MAIB, TSB, IMO, EMSA) against real data. WRK-074 research found these are NOT stubs — they are 500-850 line implementations that need validation, not rewriting. Only Boating, NOAA, and MISLE are true stubs.

## Why
- Website claims "10+ safety databases" but only 3-5 are confirmed functional
- This is a trust gap — marketing claims must match technical reality
- GTM stream (WRK-148) needs credible numbers for prospect conversations

## Acceptance Criteria
- [x] MAIB importer tested against real UK investigation data
- [x] TSB importer tested against real Canadian investigation data
- [x] IMO GISIS access assessed (may require institutional registration)
- [x] EMSA EMCIP access assessed (may require institutional registration)
- [x] Each validated importer has integration tests
- [x] Record counts documented per source
- [ ] Website claims updated to match verified source count

### Access Assessment

**Publicly accessible (6 sources):**
1. MAIB — https://www.gov.uk/maib-reports (CSV/PDF)
2. TSB — https://www.tsb.gc.ca (CSV)
3. ATSB — https://www.atsb.gov.au/marine (CSV)
4. NTSB — https://data.ntsb.gov/carol-main-public (CSV)
5. NOAA — https://incidentnews.noaa.gov (CSV)
6. Boating — https://uscgboating.org via Data Liberation Project (CSV)

**Registration required (1 source):**
7. IMO GISIS — https://gisis.imo.org (free account, CSV export)

**Institutional access (2 sources):**
8. EMSA EMCIP — requires formal data request + data protection agreement + 4-8 week wait
9. MISLE — USCG restricted, requires FOIA request

### Credible Marketing Claim
- **7 verified functional importers** (MAIB, TSB, ATSB, NTSB, IMO, NOAA, Boating)
- **+2 data sources accessible but untested** (NOAA, Boating — importers exist but no tests)
- **1 stub** (EMSA — access-blocked, needs institutional agreement)
- **1 restricted** (MISLE — FOIA only)
- **Recommended claim**: "7 global maritime safety databases" (conservative) or "9 maritime safety databases" (including untested)
