## Delivered

### `scripts/maintenance/hardware-info.sh` (new, 96 lines)
- Collects: hostname, OS, kernel, CPU (model/cores/threads), RAM, GPU (lspci + nvidia-smi),
  disks (lsblk), network interfaces
- Prints formatted SKILL.md table row at end for easy copy-paste into registry
- `--json` flag for scripting
- Executable; run on any Linux workstation

- ace-linux-1 CPU (Xeon E5-2630 v3) is Haswell-EP platform → uses DDR4, not DDR3
- Spare 32GB DDR3 ECC RAM is **not compatible** with ace-linux-1
- Best RAM fit: Windows machine (if DDR3-era board — needs `hardware-info.sh` run)
- T400 GPU: can add as second card on ace-linux-1 (if PCIe slot free) for multi-monitor
