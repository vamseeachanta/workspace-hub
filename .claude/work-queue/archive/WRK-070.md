# Import PHMSA pipeline data and build pipeline_safety module

## What
Extract the already-downloaded PHMSA ZIP files (24MB pipeline safety flagged incidents + 12 Excel files). Build the pipeline_safety module with database models and importers. Import data covering gas distribution, gas transmission, hazardous liquids, and LNG incidents.

## Why
Pipeline safety data is downloaded but sits unprocessed. The pipeline_safety module is an empty scaffold.
