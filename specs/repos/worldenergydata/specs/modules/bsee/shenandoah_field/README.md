## ğŸ“‹ Project Overview

This directory contains a comprehensive **asset lifecycle and societal economics analysis** of the **Shenandoah deepwater oil field** in the Gulf of Mexico, covering the complete cycle from discovery (2009) through first production (2025).

**Central Question:** Does distressed asset recycling create value for society and the O&G industry, or does it simply transfer wealth between investor classes?

**Key Finding:** The field demonstrates **value transfer, not value creation** at societal level. While new partners earn 37% IRR, all-in project IRR is only **4-5%** (below hurdle rate) when including $3.5B in sunk costs from original partners. Total capital deployed: **$5.9 billion** over 22 years.

## ğŸ“ Key Lessons for Deepwater Development

1. **Reserve Estimation:** Single appraisal wells insufficient; Paleogene reservoirs require conservative approach
2. **Whistleblower Value:** Early technical dissent (Lea Frye 2014) proved more accurate than management guidance
3. **Commodity Timing:** Same reservoir, different economics: Majors entered at peak costs/low prices; PE entered at trough
4. **Technology Risk:** 20k psi high-pressure development successfully deployed by independents after majors exited
5. **Governance Matters:** Disconnect between technical assessments and investor communications destroyed $1.4B in value
